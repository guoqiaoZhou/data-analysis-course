## P4 - 内容形态扩展（v1.0+）

### 网页版课程（HTML）
- **目标**：将 Markdown 知识图谱转换为可视化更强的网页课程
- **预期效果**：适配视觉学习者，支持公式渲染、交互图表、代码高亮
- **技术方案**：静态站点生成器（MkDocs / Docusaurus / VitePress）
- **内容来源**：同一套 Markdown 源文件，自动构建
- **依赖**：v1.0 文本课程质量稳定

### 播客课程（音频）
- **目标**：将「好问」对话脚本转换为播客形式
- **预期效果**：适配听觉学习者，通勤/健身时学习
- **技术方案**：AI 生成主播对话脚本 → TTS 合成 / 真人录制
- **内容来源**：GoodQuestions + DigDeeper 的追问链天然适合对话形式
- **依赖**：v0.3「好问」系统完成

### 视频课程（B站）
- **目标**：将核心知识点转换为短视频/长视频课程
- **预期效果**：触达 B 站用户群体，扩大影响力
- **技术方案**：概念文档 → 分镜脚本 → 公式可视化动画 → 案例动画
- **内容来源**：同一套知识骨架，提取关键概念和案例
- **依赖**：v1.0 内容完整 + 网页版验证公式可视化方案

### 多形态内容复用策略
```
底层：同一套知识骨架（56 个知识点 Markdown）
  ↓
中层：结构化转换
  - 网页：Markdown → HTML（静态生成）
  - 播客：GoodQuestions → 对话脚本（AI 生成）
  - 视频：概念文档 → 分镜脚本（AI 生成）
  ↓
上层：平台分发
  - 网页：自有站点 / GitHub Pages
  - 播客：小宇宙 / Apple Podcasts / Spotify
  - 视频：Bilibili / YouTube
```

**核心原则**：内容生产一次性投入，多形态复用分发。不为每种形式重新写内容，而是同一套内容的结构化转换。
