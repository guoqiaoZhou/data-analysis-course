# 双重稳健学习

## 问题引入

假设你是一家外卖平台的数据科学家。公司推出了一项"骑手奖励计划"，但奖励资格不是随机分配的——系统优先向"高潜力"骑手推送（基于历史接单量、准时率、区域供需等特征）。三个月后，运营团队问你："这个奖励计划对骑手整体接单量的因果效应是多少？"

你首先想到的是直接比较"获得奖励的骑手"和"未获得奖励的骑手"的接单量差异。但你立刻意识到这是相关关系而非因果：获得奖励的骑手本来就是高绩效者，他们的高接单量可能来自自身能力，而非奖励计划。

你决定用PSM来匹配——为每个获得奖励的骑手找一个"看起来像会获得奖励但实际没有"的骑手作为对照。但PSM只能控制可观测特征，如果存在未观测的混杂（如"骑手的努力程度"），匹配后的估计仍然偏误。

你又想到用DML框架——用机器学习估计nuisance函数，再通过正交化估计因果效应。但DML要求nuisance模型（倾向得分和结果回归）的收敛速度足够快，在实际中两个模型都可能误设。

有没有一种方法，能够像上了"双重保险"一样——即使结果模型估计错了，只要倾向得分模型正确，估计仍然可靠；即使倾向得分模型估计错了，只要结果模型正确，估计仍然可靠？这就是"双重稳健性"的核心思想：同时建模结果分布和倾向得分，只要两者之一正确设定，估计量就保持一致性。

你学过的[[stage-02-advanced/06-causal-ml/02-meta-learners/01-s-learner/S-Learner|S-Learner]]、[[stage-02-advanced/06-causal-ml/02-meta-learners/02-t-learner/T-Learner|T-Learner]]和[[stage-02-advanced/06-causal-ml/02-meta-learners/03-x-learner/X-Learner|X-Learner]]都依赖于对结果分布的建模，而双重稳健学习提供了一种"容错性最强"的估计框架——它不假设结果模型一定正确，也不假设倾向得分一定正确，而是假设"至少有一个正确"。

## 基本原理

双重稳健学习（Doubly Robust Learning）是因果推断中"容错性最强"的估计框架。其核心思想是**同时建模结果分布和倾向得分，只要两者中有一个正确设定，估计量就保持一致性**——相当于上了"双重保险"。

核心思想：将结果回归（Outcome Regression）和逆概率加权（IPW）两种估计策略结合，构造一个对模型误设更稳健的估计量。即使结果模型或倾向得分模型有一个完全错误，估计仍然可靠。

**核心公式**：

双重稳健估计量（AIPW，Augmented Inverse Probability Weighting）：

$$\hat{\tau}_{DR} = \frac{1}{n} \sum_{i=1}^n \left[ \hat{\mu}_1(X_i) - \hat{\mu}_0(X_i) + \frac{W_i(Y_i - \hat{\mu}_1(X_i))}{\hat{e}(X_i)} - \frac{(1-W_i)(Y_i - \hat{\mu}_0(X_i))}{1-\hat{e}(X_i)} \right]$$

其中：
- $\hat{\mu}_1(x), \hat{\mu}_0(x)$：结果模型估计（T-Learner 的两模型）
- $\hat{e}(x)$：倾向得分估计
- 第一项是结果回归估计，后两项是 IPW 修正项

**双重稳健性**：

若 $E[Y|X,W=1] = \hat{\mu}_1(X)$ 正确 或 $P(W=1|X) = \hat{e}(X)$ 正确（两者之一即可），则 $\hat{\tau}_{DR}$ 是 $\tau$ 的一致估计。

**与 Meta-Learner 的关系**：

- 双重稳健估计可以视为 X-Learner 和 R-Learner 的理论基础之一
- 因果森林的 Honest Estimation 也借鉴了双重稳健的思想
- 实际应用中，双重稳健常用于**平均处理效应（ATE）**估计，而 Meta-Learner 更常用于**条件平均处理效应（CATE）**估计

## 使用场景

| 场景 | 说明 | 示例 |
|---|---|---|
| 观察性研究的ATE估计 | 非随机实验，处理分配由特征决定 | 用户自发选择使用某功能，评估整体功能价值 |
| 模型设定不确定 | 不确定结果模型或倾向得分哪个更可靠 | 新场景下，两种模型都没有历史验证 |
| 需要容错性最强的估计 | 决策重要性高，不能因模型误设而偏误 | 向高层汇报的核心指标，需要统计背书 |
| 处理效应的显著性检验 | 需要构造标准误进行假设检验 | 判断某策略的整体效果是否统计显著 |
| 作为复杂方法的基准 | 评估X-Learner/R-Learner是否值得 | 若DR估计与复杂方法差异大，说明复杂方法可能过拟合 |
| 缺失数据插补 | 结果变量存在缺失，用DR进行无偏估计 | 用户流失后结果不可观测，用DR修正选择偏误 |

## 局限性

1. **倾向得分接近0或1时爆炸**：当 $\hat{e}(X_i) \approx 0$ 或 $\approx 1$ 时，IPW 项分母趋近0，估计量方差爆炸。需要截断（truncation）或重叠假设（overlap）
2. **仅直接适用于ATE，CATE扩展复杂**：标准双重稳健公式针对平均处理效应，扩展到条件处理效应需要额外的局部化或建模工作
3. **两个模型都错时失效**："双重稳健"是"两者之一正确"，不是"两者都错也稳健"。若结果模型和倾向得分模型都严重误设，估计仍偏误
4. **方差估计复杂**：标准误的构造需要考虑 nuisance 估计的变异性，简单的样本方差公式会低估真实方差
5. **高维场景下两个模型都难正确**：当特征维度极高时，结果模型和倾向得分模型都可能误设，双重稳健的优势难以发挥

## 同类算法对比

| 方法 | 适用条件 | 优势 | 劣势 |
|---|---|---|---|
| 双重稳健（DR/AIPW） | ATE估计、观察性研究、需要容错 | 双重保险、一个模型对就稳健、理论成熟 | 倾向得分极端值爆炸、CATE扩展复杂、两模型都错时失效 |
| 结果回归（G-Computation） | 结果模型可靠、随机实验 | 简单、直观、方差小 | 结果模型错则全错、无法利用倾向得分信息 |
| 逆概率加权（IPW） | 倾向得分可靠、随机实验 | 直接利用处理分配机制、模型简单 | 倾向得分极端值爆炸、仅用处理信息、忽略结果模型 |
| X-Learner | 样本不平衡、CATE估计 | CATE天然支持、利用全量信息 | 实现复杂、倾向得分误差传递 |
| R-Learner | 大样本、需要效率最优 | 渐近效率最高、正交性 | 实现最复杂、交叉拟合代价 |

## 行业使用经验

### 经验1：倾向得分截断是必做操作——我们曾被一个极端值毁掉整个估计

实践中，无论倾向得分模型多精确，都必须对极端值截断。大厂通用做法：将倾向得分限制在 $[0.05, 0.95]$ 或 $[0.1, 0.9]$ 范围内，超出范围的样本在IPW项中用边界值替代。某电商平台未做截断时，因少量样本倾向得分估计为0.001，导致DR估计量被一个样本主导，结果完全失真。截断后估计稳定且与RCT结果一致。

### 经验2：DR是观察性研究的"底线方法"——先确认ATE，再谈CATE

在无法做随机实验的场景下（如政策评估、历史数据回溯），DR是行业标准做法。经验法则是：先用DR做ATE估计，若结果显著且稳健（结果模型和倾向得分模型分别换不同设定后结论不变），再考虑用因果森林或Meta-Learner做CATE深入分析。某外卖平台评估"骑手奖励计划"的历史效果时，先用DR确认整体ATE为正，再用X-Learner拆解异质性。

### 经验3：用"模型替换检验"验证稳健性——我们曾因模型错配而误判

双重稳健的真正价值需要通过"模型替换检验"来验证：分别用不同结果模型（线性回归→梯度提升）和不同倾向得分模型（逻辑回归→随机森林）组合，看DR估计是否稳定。若不同组合下估计差异大，说明至少有一个模型严重误设，DR的稳健性未发挥。某内容平台在评估推荐算法改版时，发现DR估计在不同模型组合下差异达30%，最终判断数据不支持可靠因果推断，决策改为继续实验。

### 经验4：两个模型都错时，DR比单一方法更危险——我们曾因此过度自信

某金融场景评估"贷款额度提升"效果，结果模型和倾向得分模型都严重误设（遗漏了关键信用评分变量）。DR估计显示+15%（显著），但RCT验证真实效应仅+3%。问题在于：团队因"双重稳健"而放松了对单个模型的审视，两个模型各自有偏但方向一致，DR的"保险"失效。**教训：即使使用DR，也必须对每个 nuisance 模型做诊断（残差分析、预测AUC、校准曲线）。DR是"容错"不是"免检"。**

## 真实案例

### 案例1：（C端运营/离线分析）

**某电商平台"会员日"活动的整体效果评估**

背景：平台每月8日举办"会员日"活动（专属折扣），但参与资格由会员等级和历史消费决定（非随机），需要评估活动对整体GMV的因果效应。

做法：双重稳健估计ATE。结果模型用LightGBM估计用户GMV的条件期望，倾向得分模型用逻辑回归估计参与会员日的概率，AIPW公式计算ATE。

发现：DR估计显示会员日对参与用户的平均GMV提升+28%（95% CI: +22%~+34%）。为验证双重稳健性，分别替换结果模型为XGBoost、倾向得分模型为随机森林，估计稳定在+25%~+30%区间。

落地：确认会员日有显著正向因果效应，将活动从"每月1次"升级为"每月2次"，年度GMV增量预估3亿元。

### 案例2：（B端运营）

**SaaS平台"免费扩容"促销的续约效果分析**

背景：平台向部分企业客户推送"存储空间免费扩容"优惠，推送基于客户使用行为和合同到期时间（非随机），评估对续约率的影响。

做法：DR估计ATE。结果模型估计续约概率，倾向得分模型估计获得扩容优惠的概率，AIPW计算平均续约率提升。

发现：DR估计续约率提升+15%（CI: +10%~+20%）。但模型替换检验发现：当倾向得分模型从逻辑回归换成梯度提升后，估计变为+8%，差异显著。深入分析发现"合同到期时间"与"续约结果"强相关，但原逻辑回归未充分捕捉此非线性关系。

落地：倾向得分模型升级后重新估计，确认稳健效应约+10%，据此调整扩容优惠的推送策略，续约率提升12%。

### 案例3：（离线分析/运营）

**内容平台"创作者扶持计划"的整体效果回溯**

背景：平台过去一年向部分创作者发放"流量扶持+现金奖励"，扶持资格由算法根据创作者历史表现决定（非随机），需要回溯评估计划对平台整体内容生态的影响。

做法：DR估计ATE。结果模型估计创作者的内容产出和质量得分，倾向得分模型估计获得扶持的概率，AIPW计算平均效应。

发现：DR估计内容产出量+20%（CI: +15%~+25%），质量得分+8%（CI: +4%~+12%）。模型替换检验稳定。但进一步分析发现，质量得分的提升主要来自"低质量创作者退出"（负向选择），而非"现有创作者提升"。

落地：扶持计划继续执行，但增加"质量门槛"防止低质量创作者为获扶持而滥发内容，平台内容生态健康度提升。

## 关键词

- **双重稳健性（Doubly Robustness）**：估计量只要结果模型或倾向得分模型之一正确设定即保持一致性的性质，是DR的核心优势
- **AIPW（Augmented Inverse Probability Weighting）**：增强逆概率加权估计量，将结果回归和IPW结合的标准双重稳健估计形式
- **倾向得分截断（Propensity Score Truncation）**：将极端倾向得分值限制在合理范围内，防止IPW项分母趋近0导致方差爆炸
- **模型替换检验**：用不同模型设定重复估计，验证结果是否稳健，是检验双重稳健性是否真正发挥的实践方法
- **重叠假设（Overlap/Positivity）**：要求每个特征组合下处理组和对照组都有正概率存在，是IPW和DR的理论前提
- **Nuisance模型**：非主要关注参数但需要先行估计的模型，DR中的结果模型和倾向得分模型均属此类

## 代码示例

以下代码演示该知识点的核心概念。

```python
import numpy as np

# AIPW 估计量
# tau = mean[ D*Y/e(X) - (1-D)*Y/(1-e(X)) 
#            - (D-e(X))/[e(X)*(1-e(X))] * m(X) ]
# 其中 e(X)=P(D=1|X), m(X)=E[Y|X]
# 只要 e(X) 或 m(X) 之一正确，估计量就一致
print("双重稳健：回归对 OR IPW 对一个就够")
```
