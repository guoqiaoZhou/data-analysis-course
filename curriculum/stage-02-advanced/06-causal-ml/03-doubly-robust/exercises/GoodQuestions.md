# 好问练习 - 双重稳健学习

> 对应知识点：[[stage-02-advanced/06-causal-ml/03-doubly-robust/双重稳健学习|双重稳健学习]]

---

## Q1: 为什么需要双重稳健学习？只用PSM或只用结果回归不行吗？ [L1 概念层]

你是一家外卖平台的数据科学家。公司推出"骑手奖励计划"，但奖励资格不是随机分配的——系统优先向"高潜力"骑手推送。三个月后，运营团队问你："这个奖励计划对骑手整体接单量的因果效应是多少？"你首先想到直接比较"获得奖励的骑手"和"未获得奖励的骑手"的接单量差异，但意识到这是选择偏误。你又想到用PSM匹配，但PSM只能控制可观测特征，且在高维场景下匹配质量差。你还想到用结果回归（G-Computation）——用机器学习估计 $E[Y|X,W]$，然后计算 $\hat{\mu}_1(X) - \hat{\mu}_0(X)$ 的平均。但结果回归的问题是：如果模型误设（如遗漏了"骑手努力程度"这个未观测变量），估计就完全错误。你需要一种方法，能够像上了"双重保险"——即使结果模型错了，只要倾向得分模型正确，估计仍然可靠；即使倾向得分模型错了，只要结果模型正确，估计仍然可靠。为什么这种"双重稳健性"在观察性研究中如此重要？

## Q2: 双重稳健学习和R-Learner最容易搞混，区别在哪？ [L2 混淆层]

你团队正在讨论观察性研究的因果估计方法。有人说："双重稳健学习和R-Learner都有'正交性'——对nuisance误设容错。它们是不是一回事？"另一个人说："双重稳健是'两者之一正确就稳健'，R-Learner的Neyman正交性是'nuisance误差一阶不敏感'。两者不同。"你意识到两者的区别需要澄清。双重稳健的"双重保险"（结果模型对或倾向得分对）和R-Learner的"正交性"（nuisance误差一阶不敏感）在数学上有什么联系和区别？为什么双重稳健主要用于ATE估计，而R-Learner主要用于CATE估计？如果我要估计"平均效应"，该用双重稳健还是R-Learner？

## Q3: 初学者最常犯的错是什么？ [L3 错误层]

你刚学会双重稳健估计（AIPW），兴奋地用它评估"会员日活动"对整体GMV的因果效应。你训练了一个LightGBM结果模型和一个逻辑回归倾向得分模型，代入AIPW公式计算ATE。结果显示+28%，95% CI为[+22%, +34%]，看起来非常显著。但你没有注意到：倾向得分模型对某些样本的估计接近0（如某用户特征组合下，参与会员日的概率估计为0.003），导致IPW项 $\frac{W(Y - \hat{\mu}_1)}{\hat{e}}$ 的分母极小，该样本的IPW项异常大，几乎主导了整个ATE估计。当你将倾向得分截断到[0.05, 0.95]后，ATE变为+18%，与之前的+28%差异巨大。你意识到"倾向得分极端值"是双重稳健的致命陷阱。为什么初学者总是忽视截断操作？极端值不截断会有什么后果？

## Q4: 这个场景该用什么？ [L4 应用层]

你是一家SaaS平台的数据科学家，需要评估"免费扩容"促销对企业续约率的平均因果效应（ATE）。你有两个选择：
- 方案A：双重稳健估计（AIPW）
- 方案B：纯结果回归（G-Computation，用XGBoost估计 $E[Y|X,W]$，然后平均差分）

你的场景特征：观察性研究（非随机分配），总样本3万企业，特征维度50维，处理分配机制部分已知（基于企业使用行为和合同到期时间），结果模型和倾向得分模型都不确定哪个更可靠。为什么方案A更适合？在什么情况下你会反而选择方案B？如果倾向得分分布大量集中在0.02以下（大量企业几乎不可能获得扩容），你的选择会改变吗？

## Q5: 多个方法冲突时怎么选？ [L5 综合层]

你是一家内容平台的因果推断负责人，需要评估过去一年"创作者扶持计划"的整体效果（ATE）。场景特征：
- 观察性研究（扶持资格由算法根据历史表现决定）
- 总样本8万创作者，特征维度120维
- 需要向CEO汇报"扶持计划是否值得继续投入"
- 决策后果大（年度预算数亿元）
- 数据已固定，无法再做实验

你跑了四种ATE估计方法：
- 结果回归（G-Computation）：ATE = +25%（内容产出提升）
- IPW（逆概率加权）：ATE = +12%
- 双重稳健（AIPW）：ATE = +18%（95% CI: +12%~+24%）
- 未调整均值差：ATE = +35%

四个方法给出了差异巨大的估计（+12%到+35%）。CEO问你："到底提升了多少？25%还是12%？"你需要设计一个"观察性研究ATE估计的稳健性评估框架"，在方法冲突时给出最可靠的结论，并说明不确定性范围。你的框架是什么？在什么情况下你会建议CEO"数据不支持可靠因果推断，需要补充实验"？
