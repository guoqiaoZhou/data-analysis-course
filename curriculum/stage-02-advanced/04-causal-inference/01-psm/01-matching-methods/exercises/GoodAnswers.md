# 好问练习解答 - PSM匹配方法

> 对应知识点：[[stage-02-advanced/04-causal-inference/01-psm/01-matching-methods/PSM匹配方法|PSM匹配方法]]

---

### Q1: 为什么不能用"会员消费3500 vs 非会员1800"直接得出因果结论？

**Hints:**
- 思考"主动健身的人"和"不健身的人"的健康差异，差异来自"健身"本身还是"愿意健身的人本来就更健康"？
- 回顾[[stage-01-foundation/01-causal-inference-basics/01-potential-outcomes/潜在结果框架|潜在结果框架]]中"可忽略性"的含义

**Answer:**
直接比较是1700元的**选择性偏差**，而非因果效应。开通会员的用户在开通前就具有更高消费倾向（更活跃、更忠诚），这些**可观测混杂**使处理组和对照组不可比。PSM的核心使命是：在观察性数据中"模拟"随机实验，通过为处理组找到"在成为会员概率上相似"的对照组，消除可观测混杂带来的偏差，使比较逼近"同一个人开通/未开通会员"的平行世界。

**决策规则：** 如果处理组和对照组存在可观测的系统性差异，且无法随机化 → 先用PSM（或回归）控制混杂，再比较结果。

### Q2: PSM和回归控制变量有什么区别？

**Hints:**
- PSM是在"人"的层面找相似，回归是在"模型"的层面做调整
- 思考：当处理组和对照组特征分布差异很大时，回归的外推风险是什么？

**Answer:**
| 维度 | PSM | 回归控制 |
|---|---|---|
| 控制逻辑 | 在特征空间找相似个体配对 | 用模型系数调整组间差异 |
| 外推风险 | 无（仅在重叠区域比较） | 有（模型在数据稀疏区外推） |
| 对重叠假设 | 显式要求，天然检验 | 不检验，可能隐式外推 |
| 结果解释 | ATT（处理组平均效应） | 条件平均效应 |
| 标准误 | 需特殊处理（Abadie-Imbens/Bootstrap） | 常规标准误 |

**决策规则：**
- 如果处理组和对照组特征分布差异大、存在非重叠区域 → 用PSM，避免回归外推
- 如果特征分布重叠良好、模型设定可信 → 回归更简便，且能估计ATE
- 如果两者都不确定 → PSM+回归双重稳健（先匹配，再在匹配样本上跑回归）

### Q3: 为什么匹配后必须做平衡检验，但SMD<0.1不等于"万事大吉"？

**Hints:**
- SMD是标准化差异，除以了标准差。如果某个变量的标准差很大，原始差异可能仍很显著
- 思考：业务方关心的是"统计上平衡"还是"业务上平衡"？

**Answer:**
SMD<0.1是**统计平衡**的标准，但不等于**业务平衡**。错误在于：将SMD<0.1当作"充分条件"，忽略了原始尺度的差异。SMD = (均值差异) / 标准差，当标准差很大时，即使SMD<0.1，原始均值差异也可能有业务意义（如历史消费频次高15%）。

**正确做法：**
1. 报告SMD（标准化差异）——检验统计平衡
2. **同时报告原始尺度的均值差异**——检验业务平衡
3. 若原始差异>10%且该变量重要 → 在匹配后的回归中进一步控制该变量
4. 若关键协变量仍不平衡 → 回到倾向得分模型，加入该变量的高阶项或交互项重新估计

**决策规则：** SMD<0.1 ∧ 原始差异<10% → 平衡通过；任一不满足 → 需进一步处理。

### Q4: 评估"黑卡会员"效果时，为什么1:1最近邻匹配可能不是最佳选择？

**Hints:**
- 统计功效与样本量的关系：样本量越小，标准误越大，检验功效越低
- 思考：1:k匹配中，k增大时匹配质量和样本量的权衡

**Answer:**
1:1匹配在处理组稀缺时**浪费统计功效**。处理组500人，对照组50000人，1:1匹配仅用500个对照组样本，丢弃49500个，导致有效样本仅1000，标准误大、检验功效低。

**决策规则：**
- 处理组稀缺（处理组/对照组 < 1:5）→ 用1:k匹配，k=3-4是标准做法
- k>5后边际收益递减（额外对照组与已有对照组相似度高，信息增量小）
- k的上限：匹配质量不显著下降的前提下，通常k≤5
- 若对照组极大且处理组极小 → 考虑核匹配（利用全部对照组加权）或卡尺匹配（保证质量）

**注意：** 1:k匹配后需用聚类标准误或Bootstrap，因为同一处理组的k个匹配对照不独立。

### Q5: 倾向得分模型用XGBoost还是逻辑回归？

**Hints:**
- 倾向得分的目的是"让处理组和对照组可比"，不是"预测最准"
- 思考：复杂模型"完美分离"处理组和对照组时，对PSM意味着什么？

**Answer:**
倾向得分模型的核心目标是**产生有重叠的、可解释的倾向得分分布**，而非最大化预测准确率。

**决策规则：**
- **首选逻辑回归或浅层树模型**（最大深度≤3）→ 保留重叠区域，避免完美分离
- 若逻辑回归的AUC<0.7且组间特征差异确实非线性 → 可尝试GAM或浅层XGBoost，但**必须检查倾向得分分布重叠**
- 若用XGBoost后倾向得分分布几乎无重叠（共同支持域极小）→ **退回简单模型**，宁可牺牲预测精度，也要保证可匹配性
- 若特征维度极高且关系复杂 → 考虑[[stage-02-advanced/04-causal-inference/01-psm/02-overlap-assumption/重叠假设|重叠假设]]检验后，改用熵平衡或双重稳健估计

**本质原则：** PSM中，倾向得分是"工具"而非"目标"。工具坏了，换工具；但不要把工具打磨得过于锋利，以至于切断了处理组和对照组之间的联系。
