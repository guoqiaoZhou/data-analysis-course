# 好问练习解答 - DID平行趋势检验

> 对应知识点：[[stage-02-advanced/04-causal-inference/02-did/01-parallel-trends/DID平行趋势检验|DID平行趋势检验]]

---

### Q1: 为什么DID需要"平行趋势假设"？没有这个假设会发生什么？

**Hints:**
- DID的识别逻辑是：处理组的变化 - 对照组的变化 = 政策效应
- 思考：如果两组的变化本来就不同，"差分差"的是什么？

**Answer:**
平行趋势假设要求：如果没有政策干预，处理组和对照组的结果变化趋势应当一致。它是DID的"守门人"，因为DID的核心识别逻辑是：

$$\text{政策效应} = (Y_{\text{处理,后}} - Y_{\text{处理,前}}) - (Y_{\text{对照,后}} - Y_{\text{对照,前}})$$

这个等式成立的前提是：$Y_{\text{处理,后}}^{\text{无政策}} - Y_{\text{处理,前}} = Y_{\text{对照,后}} - Y_{\text{对照,前}}$，即处理组"如果没有政策"的变化等于对照组"实际没有政策"的变化。

**没有平行趋势的后果：**
- 处理组和对照组的事前趋势本来就不同
- DID估计量 = 真实政策效应 + 趋势差异（偏误）
- 偏误方向取决于趋势差异方向：若处理组本来改善更快，DID高估政策效应

**决策规则：** 使用DID前必须验证平行趋势。若无法验证（如事前数据不足）→ 考虑其他方法或明确标注假设未验证。

### Q2: "平行趋势检验"和"安慰剂检验"有什么区别？

**Hints:**
- 事件研究法是"直接看政策前系数是否为0"
- 安慰剂检验是"假设平行趋势成立，看虚构政策是否有效应"
- 思考："证实"和"证伪"在科学方法论中的区别

**Answer:**
| 维度 | 事件研究法（平行趋势检验） | 安慰剂检验 |
|---|---|---|
| 核心逻辑 | 直接估计政策前各期处理效应，检验是否为0 | 虚构处理时间或处理组，检验是否出现"伪效应" |
| 对平行趋势的关系 | 直接检验必要条件（事前趋势平行） | 间接检验：若平行趋势成立，伪效应不应显著 |
| 能"证实"平行趋势吗？ | 不能（只能检验必要条件，非充分条件） | 不能（只能证伪：伪效应显著→平行趋势可能不成立） |
| 能"证伪"什么？ | 政策前系数显著→平行趋势可能不成立 | 伪效应显著→DID结果可能不可靠 |
| 输出形式 | 政策前后各期系数图+置信区间 | 伪效应估计值+p值 |
| 需要的数据 | 多期面板数据（事前≥4-6期） | 可调整处理时间或处理组 |

**决策规则：**
- 有多期面板数据 → 优先做事件研究法（直接、可视化）
- 处理时间可调整 → 补充安慰剂检验（虚构时间）
- 处理组可互换 → 补充安慰剂检验（虚构处理组）
- 论文/汇报 → 两者都做，交叉验证

### Q3: 为什么"事前窗口期只有2期"做平行趋势检验是危险的？

**Hints:**
- 统计检验的功效（power）与样本量和效应量有关
- 思考：2期数据能区分"真平行"和"随机噪声"吗？

**Answer:**
错误在于混淆了"不拒绝原假设"和"原假设成立"。平行趋势检验的原假设是"政策前系数=0"（趋势平行）。2期数据时：

- **检验功效极低**：只有2个观测点，标准误大，即使存在真实的趋势差异，统计检验也很难检测到（第二类错误率高）。
- "不显著"≠"没有差异"：可能是"真的没有差异"，也可能是"差异存在但样本太少检测不到"。
- 后果：错误地认为平行趋势成立，使用DID后估计量包含趋势差异偏误。

**正确做法：**
- 事前窗口期至少4-6期（越多越好）
- 同时报告图形检验（目视判断趋势）和回归系数
- 若事前窗口期不足 → 明确声明"平行趋势假设未充分验证，结论需谨慎解读"

**决策规则：** 事前窗口期<4期 → 不宜单独依赖统计检验，需结合图形和领域知识；事前窗口期≥6期 → 统计检验+图形双重验证。

### Q4: 平行趋势检验"失败"后，是硬做DID、加交互项、还是换合成控制法？

**Hints:**
- "硬做DID"意味着忽略平行趋势失败，继续使用DID估计
- 加入交互项试图"放松"平行趋势假设，但引入了新的模型设定风险
- 合成控制法不依赖平行趋势，但需要大量对照单元

**Answer:**
| 方案 | 风险 | 适用条件 |
|---|---|---|
| **硬做DID** | 最差选择。估计量包含趋势差异偏误，结论不可信。 | 几乎不应使用 |
| **加入时间趋势交互项** | 模型设定敏感，可能过度控制。假设"趋势差异可由协变量解释"。 | 有明确可观测的混淆因素（如城市规模），且趋势差异与该因素相关 |
| **限制样本为趋势相近子集** | 丢弃部分样本，外部效度损失。 | 大部分样本趋势平行，仅少数异常单元偏离 |
| **合成控制法** | 需要大量对照单元（≥15），结果对权重敏感。 | 处理单元少，对照单元多，政策前数据长 |
| **矩阵补全/广义合成控制** | 计算复杂，结果解释性弱。 | 多处理单元，处理时间不同 |

**决策规则：**
- 平行趋势失败 + 有明确可观测混淆因素 → 加入交互项，但做模型设定敏感性分析
- 平行趋势失败 + 处理单元独特 → 换合成控制法（见[[stage-02-advanced/04-causal-inference/02-did/03-synthetic-control/合成控制法|合成控制法]]）
- 平行趋势失败 + 无替代方法 → 承认"当前数据无法支持DID"，建议补充数据或实验

### Q5: 安慰剂检验"通过"后，为什么审稿人还说"不能证明平行趋势一定成立"？

**Hints:**
- 安慰剂检验的逻辑是"如果平行趋势成立，则伪效应不应显著"
- 思考：这个命题的逆否命题是什么？逆命题成立吗？

**Answer:**
安慰剂检验的逻辑结构：
- 原命题："若平行趋势成立，则伪效应不显著"（P→Q）
- 安慰剂检验通过 = 观察到Q（伪效应不显著）
- 但"Q成立"不能推出"P成立"（逻辑上，Q→P是逆命题，不成立）

**为什么不能证实：**
- 伪效应不显著可能是因为：①平行趋势成立；②样本量小、检验功效低；③伪效应存在但未被检测到；④安慰剂设计不当（如虚构时间选择不合理）。
- 因此，"伪效应不显著"有多种解释，不能唯一归因于"平行趋势成立"。

**三重证据链构建：**
| 证据 | 角色 | 局限 |
|---|---|---|
| **事件研究法** | 直接展示政策前趋势是否平行 | 只能检验必要条件，非充分条件 |
| **安慰剂检验** | 证伪：若失败则DID不可靠 | 不能证实平行趋势 |
| **敏感性分析** | 评估"若平行趋势轻微违反，结论是否稳健" | 假设参数化 |

**决策规则：**
- 事件研究法通过 + 安慰剂检验通过 + 敏感性分析稳健 → 三重证据支持DID可信度
- 任一证据失败 → 需补充其他检验或换方法
- 三重证据都通过 → 仍不能"证明"平行趋势，但可"高度置信"地使用DID
