# DID平行趋势检验

## 问题引入

想象你是某城市交通局的数据分析师。2023年，市政府在A区实施了"限行限购"政策以缓解交通拥堵，其他区域维持现状。一年后，领导让你评估政策效果："A区的交通状况改善了吗？"

你打开数据，发现A区在政策实施后平均通勤时间从45分钟降到了38分钟。你初步判断政策有效。但同事提醒："A区可能本来就在改善，就算没政策，通勤时间也会下降。你怎么知道这7分钟的下降是政策带来的，还是趋势本身？"

这是个好问题。你想到，可以找几个没实施政策的B区、C区作为对照，看看它们的通勤时间变化。如果B区、C区的通勤时间也在下降，那A区的下降可能只是整体趋势；如果B区、C区没变化而A区下降了，那政策效果就更可信。

于是你比较了A区和对照区的变化：
- A区：政策前45分钟 → 政策后38分钟（下降7分钟）
- B区：政策前42分钟 → 政策后40分钟（下降2分钟）
- C区：政策前48分钟 → 政策后47分钟（下降1分钟）

A区下降7分钟，对照区平均下降1.5分钟，差值5.5分钟——这看起来就是政策的净效果。你兴奋地准备汇报。

但另一位同事又提出质疑："你怎么知道A区和对照区在政策前的趋势是一样的？如果A区本来就在快速改善，B区、C区本来就在停滞，那5.5分钟的差值可能不是政策效果，而是A区和对照区本来就有的趋势差异。"

这个质疑击中了要害。你学过的[[stage-01-foundation/01-causal-inference-basics/01-potential-outcomes/潜在结果框架|潜在结果框架]]告诉我们，DID方法的核心假设是"平行趋势"——如果没有政策干预，处理组和对照组的结果变化趋势应当一致。但这个假设无法直接验证，因为"没有政策干预的A区"是反事实，永远看不到。

那么，如何在有限的数据中，尽可能检验这个关键假设是否成立？如何在汇报时让领导相信"A区如果没政策，会和B区、C区走同样的趋势"？这就是平行趋势检验要解决的问题——它是DID方法可信度的"守门人"。

## 基本原理

双重差分（DID）的核心识别假设是**平行趋势假设**：处理组和对照组在没有政策干预的情况下，结果变量的变化趋势应当保持一致。

数学表达：

$$Y_{it} = \alpha + \beta D_{it} + \gamma_i + \lambda_t + \epsilon_{it}$$

其中：
- $D_{it}$：处理变量（处理组且处理后=1）
- $\gamma_i$：个体固定效应
- $\lambda_t$：时间固定效应
- $\beta$：DID估计量，即政策效应

平行趋势假设要求：$E[\epsilon_{it}|D_{it}, \gamma_i, \lambda_t] = 0$，即处理组和对照组的时间趋势不可观测部分一致。

**检验方法**：
1. **事件研究法**：引入政策前后各期虚拟变量，检验事前系数是否显著不为0
2. **安慰剂检验**：虚构处理时间或虚构处理组，检验是否出现"伪效应"
3. **图形检验**：绘制处理组和对照组的事前趋势图，目视判断

## 使用场景

| 场景 | 说明 | 示例 |
|---|---|---|
| 政策效果评估前 | 使用DID前必须验证核心假设 | 评估某城市限购政策对房价的影响前，先检验限购前各城市房价趋势是否平行 |
| 稳健性检验 | 作为DID结果可信度的支撑 | 论文审稿人质疑平行趋势时，提供图形和统计检验双重证据 |
| 模型选择 | 帮助判断DID是否适用 | 发现趋势不平行时，考虑合成控制法或加入时间趋势交互项 |
| 样本筛选 | 识别异常对照组 | 检验发现某对照组趋势偏离过大，将其剔除后重新估计 |

## 局限性

1. **无法直接检验，只能间接验证**。平行趋势是不可观测的反事实假设，事前趋势一致只是必要条件而非充分条件
2. **小样本事前期检验效力低**。若政策前仅有2-3期数据，统计检验难以拒绝原假设，但并不意味着假设成立
3. **事后趋势偏离难以解释**。即使事前平行，政策实施后两组趋势分叉可能源于政策效应，也可能源于其他冲击
4. **图形检验主观性强**。目视判断缺乏统一标准，不同研究者可能得出不同结论

## 同类算法对比

| 方法 | 适用条件 | 优势 | 劣势 |
|---|---|---|---|
| 事件研究法 | 有多期面板数据，政策实施时间明确 | 可量化动态效应，检验效力高 | 需要较长的事前窗口期 |
| 安慰剂检验（虚构时间） | 政策实施时间可调整 | 直观展示"伪政策"无效应 | 只能证伪，不能证实 |
| 安慰剂检验（虚构处理组） | 处理组和对照组可互换 | 排除选择性偏差 | 计算量大，需要多次抽样 |
| 加入协变量交互趋势 | 存在可观测的混淆因素 | 放松平行趋势假设 | 模型设定敏感，可能过度控制 |
| 合成控制法 | 处理组数量少（甚至单个） | 数据驱动选择权重，无需平行趋势 | 需要大量对照组，结果对权重敏感 |

## 行业使用经验

### 经验1：事前窗口期至少保留4-6期——我们曾被2期数据"骗"过

某大厂评估新骑手激励政策，仅使用政策前2个月数据做平行趋势检验，p值不显著就宣称假设成立。后被内部方法论团队指出：2期数据检验功效过低，无法区分"真平行"与"样本噪声"。建议至少保留4-6期事前数据，且要同时报告图形和回归系数。

### 经验2：不要只依赖图形检验——"看起来平行"不等于"统计平行"

某内容平台评估推荐算法改版对创作者活跃度的影响，PM仅凭"两条线看起来差不多"就使用DID。数据科学团队补充事件研究法回归，发现改版前第2期系数显著为负（p<0.05），提示存在事前差异。最终采用合成控制法作为稳健性检验，两种方法结论一致后才上报管理层。

### 经验3：平行趋势失败后的处理路径——不要"硬做DID"

某外卖平台评估"准时宝"保险对用户体验的影响，检验发现核心城市与非核心城市的事前订单投诉率趋势分叉。数据团队的处理路径：①加入城市-月份交互固定效应；②限制样本为趋势相近的城市子集；③改用合成控制法，以投诉率趋势相近的城市加权构造反事实。三种方法交叉验证后结论稳健。

### 经验4：安慰剂检验"通过"不是"万事大吉"——只是排除了部分威胁

某电商平台做"百亿补贴"评估，虚构上线时间为Q1，"伪效应"不显著，安慰剂检验通过。但审稿人指出：安慰剂检验只能证伪（"如果平行趋势成立，伪效应不应显著"），不能证实（"伪效应不显著不代表平行趋势一定成立"）。**建议：安慰剂检验必须配合事件研究法和敏感性分析，三者交叉验证才能增强可信度。**

## 真实案例

### 案例1：（C端运营/离线分析）

**某电商平台"百亿补贴"活动效果评估**

背景：平台于2023年Q2在部分品类上线"百亿补贴"，希望评估对GMV和复购率的因果效应。

问题：补贴品类多为高客单价商品，与非补贴品类的事前GMV趋势可能存在系统性差异。

平行趋势检验做法：
- 使用事件研究法，以Q2上线为节点，检验上线前4个季度补贴品类与非补贴品类的GMV增长率
- 回归结果显示上线前3期交互项系数均不显著（p>0.1），图形上两条趋势线基本重合
- 安慰剂检验：虚构上线时间为Q1，"伪效应"不显著

结论：平行趋势假设通过检验，DID估计结果可信。最终估计补贴对GMV提升约12%，但利润率为负，决策层据此调整了补贴策略。

### 案例2：（B端运营）

**SaaS企业"客户成功团队"介入对续约率的影响**

背景：B端SaaS公司于2023年在部分大客户试点专属客户成功（CS）团队，希望评估对年度续约率的因果效应。

问题：试点客户多为历史续约率高、ARR高的"优质客户"，存在明显选择偏差，且事前续约趋势可能不同。

平行趋势检验做法：
- 将客户按历史续约率、ARR、行业分层，在层内匹配对照组
- 绘制分层后的月度续约率趋势图，发现高ARR层中试点与对照组趋势一致，但低ARR层存在分叉
- 事件研究法回归证实：低ARR层事前第2期系数显著为正（p<0.05）

处理：剔除低ARR层样本，仅在高ARR层使用DID。同时采用PSM-DID进一步控制选择偏差。最终估计CS团队介入使续约率提升8个百分点，ROI为正，推动全量 rollout。

### 案例3：（离线分析/运营）

**网约车平台"动态定价区域"对司机供给的影响**

背景：平台于2022年在部分城市试点扩大动态定价区域，希望评估对高峰时段司机在线时长的影响。

问题：试点城市多为供需矛盾突出的头部城市，司机在线时长的事前趋势可能与对照城市不同。

平行趋势检验做法：
- 使用城市-日面板数据，以试点前6个月为事前窗口
- 图形检验显示试点城市与对照城市的事前趋势基本一致
- 但事件研究法发现：试点前1个月系数边际显著（p=0.08），提示可能存在预期效应或政策预热

处理：将政策实施时间向前调整1个月（纳入预热期），重新检验后平行趋势成立。最终估计动态定价区域扩大使高峰司机在线时长增加15%，但乘客等待时间仅下降3%，说明供给增加主要被订单增长吸收。

## 关键词

- **平行趋势假设**：DID的核心识别条件，要求处理组和对照组在政策前具有相同的时间趋势
- **事件研究法**：通过引入政策前后各期虚拟变量，动态展示政策效应并检验事前趋势的方法
- **安慰剂检验**：虚构处理时间或处理组，检验是否出现"伪效应"，用于验证DID结果稳健性
- **反事实**：处理组未受政策干预时的假想状态，DID用对照组的变化作为反事实代理
- **动态效应**：政策效应随时间变化的模式，事件研究法可估计每期处理效应
- **预期效应**：政策实施前行为主体因预期政策到来而提前改变行为，可能导致事前趋势检验失效
