### Q1 追问链: 事件研究法的基准模型中，为什么k=-1期的系数被约束为0？

**追问1:** 事件研究法的基准模型中，$D_{it}^k$ 是相对政策实施时间的第k期虚拟变量，k=-1为基准期，系数约束为0。为什么必须约束基准期？如果不约束会发生什么？
**Answer:** 基准期约束是为了**避免完全多重共线性**。事件研究模型中包含了个体固定效应 $\mu_i$ 和时间固定效应 $\lambda_t$，如果同时引入所有相对时间虚拟变量（包括k=-1），则这些虚拟变量与固定效应存在线性依赖关系——所有相对时间虚拟变量的和等于1（或时间固定效应的线性组合），导致设计矩阵不满秩。约束k=-1系数为0，相当于以政策前1期为参照点，其他期的系数表示"相对于政策前1期的差异"。这与DID中"政策后 vs 政策前"的比较逻辑一致。

**追问2:** 基准期约束为0后，政策前各期系数的经济学含义是什么？如果政策前3期系数为-0.5（p=0.3），这意味着什么？
**Answer:** 政策前k期系数表示："在政策前k期，处理组和对照组的结果差异相对于基准期的变化"。若政策前3期系数为-0.5（不显著），意味着："在政策前3期，处理组和对照组的趋势差异相对于政策前1期没有显著变化"——这是平行趋势成立的证据。注意：政策前系数检验的是"趋势是否平行"，而非"水平是否相同"（水平差异由固定效应控制）。若政策前某期系数显著为正 → 提示该期处理组相对对照组有异常上升，平行趋势可能不成立。

**追问3:** 如果政策实施时间存在异质性（不同单元在不同时间处理），基准期应该如何设定？传统方法的基准期设定为什么会产生问题？
**Answer:** 统一处理时间下，所有单元的k=-1是同一日历时间，基准期明确。交错处理下，不同单元的"政策前1期"对应不同日历时间，传统方法将"相对时间"对齐后，同一k值对应不同日历时间，导致：①时间固定效应 $\lambda_t$ 与相对时间虚拟变量混淆；②"从未处理组"的基准期不明确（它们没有"政策前1期"）。CS方法的处理方式是：为每个处理时间-队列单独定义基准期，或使用"从未处理组"的日历时间作为参照。见[[stage-02-advanced/04-causal-inference/02-did/01-parallel-trends/DID平行趋势检验|DID平行趋势检验]]中关于交错DID的讨论。

### Q2 追问链: 事件研究法的"动态效应"和"脉冲响应函数"有什么区别？

**追问1:** 事件研究法估计的政策后系数可以看作"脉冲响应"——政策冲击后各期的效应。但它和宏观经济学中的"脉冲响应函数"（Impulse Response Function, IRF）有什么区别？
**Answer:** 核心区别在于**模型结构和识别假设**：事件研究法基于DID框架，识别假设是平行趋势——处理组和对照组在没有政策时趋势相同，政策后的差异归因于政策。脉冲响应函数通常基于VAR或局部投影（Local Projection）框架，识别假设是"外生冲击"——冲击与误差项不相关。事件研究法的"动态效应"是"处理组 vs 对照组"的差异随时间变化；IRF是"变量对冲击的反应"随时间变化。事件研究法需要面板数据和处理组/对照组结构；IRF需要时间序列数据和冲击的外生性假设。

**追问2:** 局部投影法（Local Projection, LP）可以作为事件研究法的替代吗？它的优势和劣势是什么？
**Answer:** LP由Jordà（2005）提出，直接回归：$Y_{i,t+h} - Y_{i,t-1} = \alpha^h + \beta^h D_{it} + \gamma^h X_{it} + \epsilon_{i,t+h}$，对每个h单独估计。优势：①**无需指定动态滞后结构**（不像VAR需要指定滞后阶数）；②**更灵活**，可处理异质性处理效应；③**标准误计算简单**。劣势：①**估计效率低**（每期h单独回归，不利用跨期信息）；②**系数可能非单调**（因为没有跨期约束）；③**长h时样本量减少**。决策规则：若关心"灵活的脉冲响应形状"且样本量大 → LP；若需要统计效率且模型设定可信 → 事件研究法。

**追问3:** 在什么情况下，事件研究法的动态系数图可能出现"政策后系数先负后正"？这种模式该如何解读？
**Answer:** "先负后正"模式的可能机制：①**适应期成本**（如限行初期司机不熟悉路线，拥堵加剧；后期适应后改善）；②**资源重新配置**（如政策初期资源从A领域撤出导致短期下降，后期资源投入B领域带来长期增长）；③**竞争反应**（如平台政策初期商家观望/退出，后期新商家进入填补）；④**预期修正**（如政策初期市场过度悲观，后期修正）。解读时必须结合业务机制，不能仅凭系数图下结论。建议：出现反直觉模式时，做机制分析（如分行业、分用户群的事件研究法），识别哪个子群体驱动了该模式。

### Q3 追问链: 事件研究法中"预期效应"和"政策效应"如何区分？

**追问1:** 如果政策实施前，市场已预期政策到来并提前改变行为（如预期限行的城市提前买车），事件研究法的政策前系数会显示什么？如何与"平行趋势不成立"区分？
**Answer:** 预期效应会导致政策前1期（或前2-3期）系数显著为正/负（取决于预期方向）。这与"平行趋势不成立"在统计上表现相同——政策前系数显著。但两者的本质不同：①**平行趋势不成立**是"处理组和对照组本来就有不同趋势"（结构性差异）；②**预期效应**是"处理组因预期政策而改变行为"（暂时性反应）。区分方法：①**业务知识**：调查是否存在政策预热、媒体报道、内部泄露等预期渠道；②**系数模式**：预期效应通常集中在政策前1-2期，且方向与政策效应一致；平行趋势差异通常是持续的、方向不确定的；③**调整基准期**：将基准期前移至预期效应开始之前，若政策前系数变为不显著 → 确认是预期效应而非趋势差异。

**追问2:** 预期效应对政策效应估计有什么影响？如果不处理，DID估计会偏往哪个方向？
**Answer:** 预期效应会使DID估计**偏误方向不确定**，取决于预期行为的方向：①若预期导致"提前释放需求"（如预期补贴而提前消费）→ 政策前处理组已经"高"了，政策后DID估计偏低（因为部分效应已被预期消化）；②若预期导致"观望/推迟"（如预期降价而推迟购买）→ 政策前处理组"低"了，政策后DID估计偏高（因为政策后包含"被压抑需求的释放"）。处理策略：①将政策实施时间前移至"预期效应开始"的时间点；②在模型中显式控制预期效应（如加入"预期期"虚拟变量）；③若预期效应无法量化 → 在汇报中声明"估计可能包含预期效应，真实政策效应可能偏离"。

**追问3:** 在什么情况下，预期效应本身就是研究对象（而非干扰项）？
**Answer:** 当研究问题是"政策 announcement 的影响"而非"政策 implementation 的影响"时。例如：①评估"反垄断调查公告"对平台股价的即时影响——公告本身就会改变市场预期和行为，不需要等到调查结束；②评估"加息预期"对房地产市场的影响——预期本身就会改变购房决策。此时，事件研究法的"政策前"系数恰恰是研究对象，"政策后"系数则是预期实现后的额外效应。识别策略：以"公告日"而非"实施日"为事件点，重新估计事件研究法。
