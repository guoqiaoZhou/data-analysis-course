# 好问练习解答 - 合成控制法

> 对应知识点：[[stage-02-advanced/04-causal-inference/02-did/03-synthetic-control/合成控制法|合成控制法]]

---

### Q1: 为什么DID找不到合适对照组时，合成控制法能救场？

**Hints:**
- DID要求"存在一个合适的对照组"（平行趋势假设）
- 合成控制法不找单一相似单元，而是让数据自己说话
- 思考：多个"不太相似"的单元加权组合，能否比一个"相似"的单元更接近处理单元？

**Answer:**
合成控制法的核心思想：**用多个对照单元的加权组合构造"虚拟处理单元"**，而非寻找单一相似对照。

**为什么能救场：**
- DID的瓶颈：处理单元独特时，任何单一对照单元都不合适
- 合成控制法的突破：多个不太相似的单元（如二线城市A + 三线城市B + 资源型城市C）加权后，可能在"结果趋势"和"关键特征"上与处理单元高度匹配
- 数学保证：凸组合约束（$w_j \geq 0, \sum w_j = 1$）确保合成单元在政策前不会外推到观测范围之外

**可比性保证：**
- 政策前MSPE（均方预测误差）低 → 合成单元在政策前与处理单元高度匹配
- 政策后差异 → 归因于政策（假设政策后对照单元行为模式不变）

**决策规则：**
- 处理单元独特、找不到单一合适对照 → 合成控制法
- 处理组和对照组均多、趋势可平行 → DID更简便
- 有面板数据、政策前时期≥12期、对照单元≥15个 → 合成控制法可行

### Q2: 合成控制法和DID的"加权DID"有什么区别？

**Hints:**
- 合成控制法的权重由算法自动确定（最小化MSPE）
- 加权DID的权重由研究者主观设定（如按GDP、人口）
- 思考：数据驱动 vs 主观设定的本质区别是什么？

**Answer:**
| 维度 | 合成控制法 | 加权DID |
|---|---|---|
| 权重来源 | **数据驱动**（最小化政策前MSPE） | **研究者主观设定**（如GDP、人口） |
| 权重约束 | $w_j \geq 0, \sum w_j = 1$（凸组合） | 通常无约束，可正可负 |
| 识别假设 | 政策前合成单元与处理单元匹配 | 处理组与加权对照组趋势平行 |
| 外推风险 | 低（凸组合保证内插） | 可能高（负权重导致外推） |
| 可解释性 | 权重揭示哪些单元贡献最大 | 权重反映研究者先验 |
| 标准误 | 需置换检验 | 解析标准误可得 |

**决策规则：**
- 对照单元多、特征复杂、研究者不确定权重 → 合成控制法（数据说话）
- 对照单元少、有明确理论指导权重 → 加权DID（如按经济规模加权）
- 需要可解释的标准误 → 加权DID
- 处理单元独特、任何单一权重都不合适 → 合成控制法

### Q3: 为什么MSPE高时，合成控制法的政策后效应估计不可靠？

**Hints:**
- MSPE = 政策前处理单元与合成单元的均方预测误差
- 思考：如果政策前合成单元都拟合不好，凭什么相信政策后它能作为"反事实"？

**Answer:**
MSPE是合成控制法的"生命线"，因为它衡量**政策前合成质量**。高MSPE意味着：

- 政策前处理单元与合成单元差距大
- 合成单元未能成功"复现"处理单元的反事实路径
- 政策后两者的差异可能源于"合成失败"而非"政策效应"

**MSPE诊断标准：**
| MSPE水平 | 含义 | 建议 |
|---|---|---|
| MSPE低（如<1%的 outcome 均值） | 合成质量好 | 政策后效应可信 |
| MSPE中等（如1%-5%） | 合成质量一般 | 效应估计需谨慎，需多重稳健性检验 |
| MSPE高（如>5%） | 合成质量差 | 效应估计不可信，需换方法或重新设计 |

**诊断方法：**
1. 画"实际 vs 合成"时间序列图，目视政策前拟合度
2. 计算MSPE比率 = 政策后MSPE / 政策前MSPE
3. 与安慰剂检验中所有对照单元的MSPE比率比较排名
4. 若处理单元的MSPE比率在对照单元中排名靠前（如Top 5%）→ 效应显著

**决策规则：**
- 政策前MSPE高 → 不报告政策后效应，先优化合成（增加对照单元、延长政策前期、加入更多特征）
- 政策前MSPE低但MSPE比率排名不靠前 → 效应不显著，不能排除随机波动

### Q4: 合成控制法"无法提供解析标准误"，怎么判断效应是否显著？

**Hints:**
- 传统SCM没有解析标准误公式
- 置换检验（安慰剂检验）是推断显著性的主要方法
- 思考：为什么"将每个对照单元轮流作为假处理单元"能帮助我们判断显著性？

**Answer:**
**置换检验（Fisher's Exact Test思路）：**

1. **构造安慰剂分布**：将每个对照单元轮流作为"假处理单元"，用其余对照单元构造合成控制，计算其政策后MSPE
2. **计算MSPE比率**：对每个"假处理单元"，计算 政策后MSPE / 政策前MSPE
3. **排名比较**：将真实处理单元的MSPE比率与所有"假处理单元"的比率排序
4. **p值**：真实处理单元的排名百分位即为p值（如排名Top 5% → p≈0.05）

**判断标准：**
- 真实处理单元的MSPE比率在所有对照单元中排名靠前（如Top 5%或Top 10%）→ 效应显著
- 排名居中 → 效应不显著，不能排除随机波动
- 排名靠后 → 效应为负或不显著

**注意事项：**
- 对照单元数量需≥15，否则置换检验功效低
- 若某对照单元政策前MSPE极高（合成失败），其政策后MSPE比率可能异常高，需排除异常值
- 报告排名和比率，而非仅报告"p<0.05"

**决策规则：**
- 对照单元≥15 + 真实处理单元MSPE比率排名Top 5% → 效应显著
- 对照单元<15 → 置换检验功效不足，需谨慎解读
- 多重稳健性检验（排除最高权重单元、使用不同时期估计权重）方向一致 → 增强信心

### Q5: 合成控制法显示"DAU下降18%"，权重集中在邻国A（60%）和邻国B（35%）。汇报时该强调效应值还是权重解释？

**Hints:**
- 合成控制法的可信度不仅取决于效应值，还取决于"合成单元是如何构造的"
- 思考：如果权重集中在两个与处理单元"不相似"的国家，18%的效应还可信吗？

**Answer:**
**权重解释比数字更重要。** 合成控制法的核心可信度来自"合成单元的构造合理性"，而非效应值本身。

**汇报策略：**
1. **先解释权重**："合成该国由邻国A（60%）和邻国B（35%）加权构成，两国均为同语系、同发展阶段、同互联网渗透率，政策前DAU趋势高度一致"
2. **再报告效应**："政策实施后，实际DAU比合成DAU低18%"
3. **最后给显著性**："MSPE比率在所有对照单元中排名Top 3%，p≈0.03"

**为什么权重解释优先：**
- 权重揭示了"哪些单元对反事实构造贡献最大"
- 若权重集中在"理论上不相似"的单元 → 效应值不可信
- 若权重集中在"理论上相似"的单元 → 效应值可信度高
- 权重稀疏（集中在少数单元）是正常现象，但需解释这些单元的相似性

**效应值可独立汇报的条件：**
- 政策前MSPE极低（拟合几乎完美）
- 多重稳健性检验（排除最高权重单元、改变目标函数）结果一致
- 有外部验证（如其他方法也得到类似估计）

**决策规则：**
- 汇报时 → 先权重解释，后效应值，最后显著性
- 若权重难以解释（如权重集中在"不相似"单元）→ 承认合成质量不足，不强行报告效应值
- 若权重合理且多重检验通过 → 可自信报告效应值
