## 一、因果推断核心公式

### 1.1 潜在结果框架

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $\tau_i = Y_i(1) - Y_i(0)$ | 个体处理效应（ITE） | [[stage-01-foundation/01-causal-inference-basics/01-potential-outcomes/潜在结果框架\|潜在结果框架]] |
| $\tau_{ATE} = E[Y(1) - Y(0)]$ | 平均处理效应 | 潜在结果框架 |
| $\tau_{ATT} = E[Y(1) - Y(0) \| D=1]$ | 处理组平均处理效应 | 潜在结果框架 |
| $\tau(x) = E[Y(1) - Y(0) \| X=x]$ | 条件平均处理效应（CATE） | 潜在结果框架 |
| $Y_i = D_i \cdot Y_i(1) + (1-D_i) \cdot Y_i(0)$ | 观测结果方程 | 潜在结果框架 |
| $(Y(1), Y(0)) \perp D \,\|\, X$ | 条件独立性假设 | 潜在结果框架 |
| $0 < P(D=1\|X=x) < 1, \quad \forall x$ | 重叠假设 | 潜在结果框架 |

### 1.2 随机化实验

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $D \perp (Y(1), Y(0))$ | 随机化独立性 | [[stage-01-foundation/01-causal-inference-basics/02-randomized-experiments/随机化实验原理\|随机化实验原理]] |
| $\hat{\tau} = \bar{Y}_{treat} - \bar{Y}_{control}$ | ATE估计量 | 随机化实验原理 |
| $Var(\hat{\tau}) = \frac{\sigma^2_{treat}}{n_{treat}} + \frac{\sigma^2_{control}}{n_{control}}$ | 估计量方差 | 随机化实验原理 |
| $Y_i = \alpha + \tau D_i + \epsilon_i$ | 回归形式 | 随机化实验原理 |

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $e(X) = P(D=1\|X)$ | 倾向得分定义 | [[stage-02-advanced/04-causal-inference/01-psm/01-matching-methods/PSM匹配方法\|PSM匹配方法]] |

### 1.6 DML（双重机器学习）

## 二、统计基础公式

### 2.1 t检验

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$ | 单样本t统计量 | [[stage-01-foundation/02-statistical-basics/01-t-test/t检验\|t检验]] |
| $s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}$ | 合并方差 | t检验 |
| $t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{s_p^2(\frac{1}{n_1}+\frac{1}{n_2})}}$ | 独立样本t（方差齐性） | t检验 |
| $t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$ | Welch's t检验（方差不齐） | t检验 |
| $t = \frac{\bar{d}}{s_d / \sqrt{n}}$ | 配对t检验 | t检验 |
| $t = \frac{\text{效应}}{SE}$ | t统计量通用形式 | t检验 |

### 2.2 卡方检验

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$ | 卡方统计量 | [[stage-01-foundation/02-statistical-basics/02-chi-square-test/卡方检验\|卡方检验]] |
| $E_{ij} = \frac{(\text{行合计}_i)(\text{列合计}_j)}{\text{总样本量}}$ | 期望频数 | 卡方检验 |

### 2.3 OLS回归

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $\min_{\beta} \sum_{i=1}^n (Y_i - \beta_0 - \beta_1 X_{i1} - ... - \beta_k X_{ik})^2$ | 最小二乘目标 | [[stage-01-foundation/02-statistical-basics/04-ols-regression/OLS回归\|OLS回归]] |
| $\hat{\beta} = (X'X)^{-1}X'Y$ | 解析解（矩阵形式） | OLS回归 |
| $E[\epsilon\|X] = 0$ | 外生性假设 | OLS回归 |

### 2.4 Logistic回归

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $P(Y=1\|X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + ... + \beta_k X_k)}}$ | Sigmoid概率 | [[stage-01-foundation/02-statistical-basics/05-logistic-regression/Logistic回归\|Logistic回归]] |
| $\log\frac{P}{1-P} = \beta_0 + \beta_1 X_1 + ... + \beta_k X_k$ | Logit变换 | Logistic回归 |

### 2.5 固定效应模型

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $Y_{it} = \alpha_i + \beta X_{it} + \epsilon_{it}$ | FE模型 | [[stage-01-foundation/02-statistical-basics/06-fixed-effects/固定效应模型\|固定效应模型]] |
| $Y_{it} - \bar{Y}_i$ | 组内去均值化 | 固定效应模型 |

### 2.6 检验力与样本量

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $n = \frac{16 \sigma^2}{\delta^2}$ | 样本量近似公式（α=0.05, Power=80%） | [[stage-01-foundation/02-statistical-basics/07-power-analysis/检验力分析与样本量计算\|检验力分析]] |
| $n = 2\left(\frac{(Z_{1-\alpha/2} + Z_{1-\beta})\sigma}{\delta}\right)^2$ | 样本量精确公式 | 检验力分析 |

### 2.7 CUPED方差缩减

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $\tilde{Y} = Y - \hat{\theta}(X - \bar{X})$ | CUPED调整公式 | [[stage-01-foundation/03-ab-testing/04-cuped/CUPED方差缩减\|CUPED方差缩减]] |
| $Var(\tilde{Y}) = Var(Y)(1 - \rho^2)$ | 方差缩减效果 | CUPED方差缩减 |
| $\hat{\theta} = Cov(Y,X) / Var(X)$ | 最优系数 | CUPED方差缩减 |

### 2.8 后分层分析

| 公式 | 含义 | 对应知识点 |
|------|------|-----------|
| $\hat{\tau}_{post} = \sum_{s=1}^S w_s \hat{\tau}_s$ | 后分层估计量 | [[stage-01-foundation/03-ab-testing/05-post-stratification/后分层分析\|后分层分析]] |

## 五、符号速查表

| 符号 | 含义 |
|------|------|
| $Y_i(1), Y_i(0)$ | 个体$i$接受/未接受处理的潜在结果 |
| $D_i$ / $W_i$ / $T_i$ | 处理指示变量（不同章节习惯不同） |
| $\tau$ | 处理效应 |
| $ATE$ | 平均处理效应（Average Treatment Effect） |
| $ATT$ | 处理组平均处理效应（Average Treatment Effect on Treated） |
| $CATE$ | 条件平均处理效应（Conditional ATE） |
| $X$ | 协变量/特征向量 |
| $e(X)$ | 倾向得分（Propensity Score） |
| $g(X), m(X)$ | Nuisance函数（结果模型、处理模型） |
| $\psi$ | 得分函数（Score Function） |
| $\rho$ | 相关系数 |
| $\alpha$ | 显著性水平（通常0.05） |
| $\beta$ | 第二类错误概率（通常0.2） |
| $1-\beta$ | 检验力/功效（Power，通常0.8） |
| $\delta$ | 最小可检测效应（MDE） |
| $SE$ | 标准误（Standard Error） |
| $SMD$ | 标准化均值差（Standardized Mean Difference） |
| $WOE$ | 证据权重（Weight of Evidence） |
| $IV$ | 信息值（Information Value） |
| $\phi_j$ | SHAP值（特征$j$的贡献） |
| $ALE$ | 累积局部效应（Accumulated Local Effects） |
