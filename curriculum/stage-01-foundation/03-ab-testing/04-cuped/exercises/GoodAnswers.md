# 好问练习解答 - CUPED方差缩减

> 对应知识点：[[stage-01-foundation/03-ab-testing/04-cuped/CUPED方差缩减|CUPED方差缩减]]

---

### Q1: 为什么样本量不够时，"看起来有效果"却不能说"统计显著"？

**Hints:**
- 统计显著性的公式是什么？和样本量、效应量、方差有什么关系？
- "噪声"在统计中对应什么概念？
- CUPED的方差缩减公式 $Var(\tilde{Y}) = Var(Y)(1 - \rho^2)$ 告诉我们什么？

**Answer:**
统计显著性取决于三个因素：效应量、样本量、方差。公式上，t统计量 = 效应量 / 标准误，标准误 = 标准差 / √n。当样本量固定（如300商家）、效应量固定（+5pp）时，方差越大，标准误越大，t统计量越小，p值越大。

CUPED的核心原理：利用实验前同一指标的历史值作为协变量，对实验期间的指标做线性调整，从而缩减方差。

$$\tilde{Y} = Y - \hat{\theta}(X - \bar{X})$$

**方差缩减效果**：
$$Var(\tilde{Y}) = Var(Y)(1 - \rho^2)$$

若 $\rho=0.7$，方差缩减约50%，标准误缩减约30%，相当于样本量翻倍的效果。

**决策规则**：
- 如果有可靠的历史数据，且 $\rho > 0.5$ → 使用CUPED
- 如果没有历史数据，或 $\rho < 0.5$ → CUPED效果有限，优先考虑增加样本量或延长实验周期
- CUPED不改变点估计（效应量仍是+5pp），只改变精度（标准误变小，p值变小）

### Q2: CUPED和回归调整（ANCOVA）是一回事吗？

**Hints:**
- CUPED的 $\hat{\theta}$ 是怎么估计的？只基于对照组，还是基于全部数据？
- 回归调整中，处理变量T和协变量X的交互项怎么处理？
- 在什么条件下，两者会给出相同的结果？

**Answer:**
**不是一回事，但数学上密切相关。**

| 维度 | CUPED | 回归调整（ANCOVA） |
|---|---|---|
| **核心公式** | $\tilde{Y} = Y - \hat{\theta}(X - \bar{X})$ | $Y = \beta_0 + \beta_1 T + \beta_2 X + \epsilon$ |
| **系数估计** | $\hat{\theta}$ 基于对照组数据 | $\beta_2$ 基于全部数据 |
| **处理效应** | 比较调整后的 $\tilde{Y}$ | 直接读取 $\beta_1$ |
| **假设** | 协变量与处理独立 | 协变量与处理独立（随机化保证） |
| **灵活性** | 单一协变量，简单 | 可多协变量，灵活 |

**关键区别**：
- CUPED的 $\hat{\theta}$ 只在对照组上估计，避免处理影响协变量系数
- 回归调整若用全部数据估计 $\beta_2$，处理效应会"泄漏"到协变量系数中
- 大样本下、随机化完美时，两者结果趋同；小样本或协变量不平衡时，CUPED更稳健

**决策规则**：
- 如果只有一个强相关协变量（如实验前同一指标）→ CUPED更简洁、更稳健
- 如果需要控制多个协变量 → 回归调整更灵活，但需确保模型设定正确
- 如果样本量小（<500）→ 优先CUPED，回归调整过拟合风险更高

### Q3: 为什么CUPED后效应量不变，但有人却汇报"效应翻倍了"？

**Hints:**
- CUPED改变的是点估计还是标准误？
- p值从0.12变为0.03，意味着什么？
- 业务方的直觉"p变小=效果变强"错在哪？

**Answer:**
**CUPED不改变效应量，只改变估计精度。**

**事实**：
- 原始分析：效应 = +3.2元，标准误 = 1.8，p = 0.12
- CUPED分析：效应 = +3.2元，标准误 = 1.2，p = 0.03
- 效应量始终是+3.2元，变化的是标准误（从1.8降至1.2）

**为什么这个错误常见**：
1. **直觉陷阱**：p值变小，业务方直觉上觉得"效果更强了"
2. **汇报模糊**：分析师说"现在显著了"，没明确说"效应量没变，只是更确定了"
3. **激励机制**：显著的结果更容易推动，分析师可能无意识地"包装"结论

**正确表述**：
- "CUPED后估计精度提升，同一效应量（+3.2元）现在以更高置信度被检测到。"
- "标准误从1.8降至1.2，相当于有效样本量增加了约1.8²/1.2² = 2.25倍。"

**决策规则**：
- 汇报CUPED结果时，必须同时报告：原始效应量、CUPED后效应量、标准误变化、方差缩减比例
- 如果原始效应量和CUPED后效应量差异很大 → 检查协变量是否被处理污染
- 绝对禁止：把"显著性变化"包装成"效应变强"

### Q4: 新用户实验没有历史数据，能用CUPED吗？如果不能，该用什么？

**Hints:**
- CUPED的核心假设是什么？"实验前同一指标"意味着什么？
- 新用户为什么"天然"不适合CUPED？
- 分层随机化和CUPED分别在什么阶段发挥作用？

**Answer:**
**新用户实验不适合CUPED，应使用分层随机化。**

**原因**：
1. **无历史数据**：CUPED需要"实验前同一指标"，新用户没有历史GMV、历史活跃度
2. **替代协变量相关性低**：用"注册后1小时浏览数"替代，ρ通常<0.5，方差缩减有限
3. **协变量不稳定**：新用户的行为模式尚未稳定，实验前数据与实验期间相关性弱

**正确方案：分层随机化**

| 阶段 | 方法 | 作用 |
|---|---|---|
| **设计阶段** | 分层随机化 | 按注册渠道、设备类型、地域分层，保证层内平衡 |
| **分析阶段** | 后分层分析 | 按层加权汇总，消除层间比例不均 |

**决策规则**：
- 新用户实验 → 默认采用分层随机化，不指望CUPED
- 老用户实验 → 优先评估CUPED可行性（ρ>0.5则使用）
- 混合实验（新老用户都有）→ 对老用户用CUPED，对新用户用分层随机化，分别报告

### Q5: 相关系数ρ=0.3时，CUPED方差缩减9%，业务方要求据此全量，你怎么办？

**Hints:**
- ρ=0.3时，方差缩减9%，标准误缩减多少？对p值的影响有多大？
- "边缘显著"（p=0.07）和"不显著"（p=0.12）在决策上有什么区别？
- 统计可靠性和业务紧迫性之间的权衡，你的底线在哪？

**Answer:**
**拒绝全量。ρ<0.5时，CUPED不应作为决策依据。**

**数学事实**：
- ρ=0.3 → 方差缩减 = 1 - 0.3² = 9%
- 标准误缩减 = √9% ≈ 5%（几乎无变化）
- p从0.12变为0.07，只是"噪音级别的改善"，不改变统计推断的可靠性

**为什么拒绝**：
1. **CUPED的降噪效果有限**：ρ<0.5时，方差缩减<25%，标准误缩减<13%，不足以改变"不显著"的本质
2. **边缘显著不是显著**：p=0.07仍大于α=0.05，统计上仍不能拒绝"无效应"的原假设
3. **业务方可能事后选择**：如果p=0.07就全量，下次p=0.08、0.09也会有人要求全量，标准逐渐崩坏

**判断标准**：
- **ρ ≥ 0.5**：CUPED方差缩减≥25%，可考虑作为辅助分析，但决策仍以原始分析为主
- **ρ ≥ 0.7**：CUPED方差缩减≥50%，标准误显著降低，可将CUPED结果作为主要参考
- **ρ < 0.5**：CUPED效果有限，不应改变原始分析结论

**替代方案**：
- 扩大样本量（如从300增至600）
- 延长实验周期（从2周增至4周）
- 收集更多定性反馈，补充定量分析
- 若业务紧迫，可小规模试点（如10%流量），但明确声明"基于探索性分析，非统计显著结论"
