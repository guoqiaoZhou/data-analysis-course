# 好问练习解答 - t检验

> 对应知识点：[[stage-01-foundation/02-statistical-basics/01-t-test/t检验|t检验]]

---

### Q1: 为什么"实验组18.5% vs 对照组15.2%"不能直接说"活动有效"？

**Hints:**
- 思考"随机波动"的含义：如果活动其实无效， rerun一次实验差异会是多少？
- 回忆抽样分布：样本均值差异的分布是什么？
- t检验的零假设是什么？

**Answer:**
"有差距"是样本层面的观测，"显著差异"是对总体层面的推断。即使活动真实无效（总体转化率相同），由于抽样随机性，实验组和对照组的样本转化率也会有差异——这个差异的分布就是抽样分布。t检验回答的是："在活动真实无效的假设下，观察到3.3pp或更大差异的概率是多少？"

如果p<0.05，意味着"活动无效时看到这么大差异的概率<5%"，我们有理由拒绝"活动无效"的假设。但p>0.05时，"不能拒绝无效"≠"活动有效"。

**决策规则：**
- 样本差异存在 → 必须做假设检验才能推断总体
- p<0.05 → 在5%显著性水平下拒绝"无效应"，认为差异"统计显著"
- p>0.05 → 不能拒绝"无效应"，差异可能由随机波动造成
- 任何"看起来有差距"的结论 → 必须经过检验，不能用肉眼判断

### Q2: t检验和Mann-Whitney U检验有什么区别？

**Hints:**
- 思考t检验的"均值"对异常值的敏感度
- Mann-Whitney U检验的"分布位移"是什么意思？
- 如果关心的是"平均效应"vs"整体分布"，哪个方法更合适？

**Answer:**
**t检验**：比较两组**均值差异**，假设数据近似正态（或大样本下中心极限定理保护）。对异常值敏感——一个消费10万的超级用户会显著拉高实验组均值，导致"显著"结论，但这个"显著"不代表大多数用户的行为改变。

**Mann-Whitney U**：比较两组**分布的随机占优关系**（stochastic dominance），不假设正态分布，对异常值稳健。但它检验的不是"均值差异"，而是"一个组的值倾向于大于另一个组"。如果两组分布形状不同（如实验组方差更大），Mann-Whitney U可能检测到"分布差异"而非"均值差异"。

**实验组有超级用户的场景：**
- t检验：可能被异常值驱动，p<0.05但中位数无变化
- Mann-Whitney U：不受异常值影响，但如果分布形状不同，结果难解释

**决策规则：**
- 数据近似正态、关心均值、无极端异常值 → t检验
- 数据严重偏态/有异常值、关心"典型用户" → Mann-Whitney U或中位数检验
- 大样本（>1000）→ t检验对轻度非正态稳健，可用
- 业务决策关心"平均效应"（如总GMV）→ t检验更合适；关心"典型用户体验" → 非参数更优
- 任何情况下 → 同时报告均值和中位数，避免单一指标误导

### Q3: 为什么p=0.051不等于"边缘显著"？

**Hints:**
- 思考p值的定义：在零假设下观察到当前或更极端结果的概率
- 0.051和0.049在数学上有什么区别？
- "调阈值"从0.05改为0.10，在统计上做了什么？

**Answer:**
p=0.051意味着"在零假设下，观察到当前差异或更大差异的概率是5.1%"；p=0.049意味着"这个概率是4.9%"两者在数学上几乎没有区别——差异仅0.2个百分点，完全在抽样波动的范围内。称p=0.051为"边缘显著"是连续化解读p值，但p值本身是**二值决策工具**（拒绝/不拒绝），不是"置信度度量"。

"调阈值"的问题：
1. **事后选择阈值**：看到p=0.051后改为0.10，是"数据窥探"（data snooping）， inflate假阳性率
2. **阈值是约定**：0.05是领域惯例，不是物理定律。但一旦设定，不应事后更改
3. **正确的做法**：如果0.05是标准，p=0.051就是不显著；如果业务能接受10%假阳性率，应在分析**前**设定α=0.10

初学者错误原因：
- 把p值当作"效应真实性的概率"（不是，它是"数据在零假设下的概率"）
- 渴望得到"显著"结论，对接近阈值的p值做"善意解读"
- 不理解阈值是"决策规则"而非"真实性的连续度量"

**决策规则：**
- p<α → 拒绝零假设，差异"统计显著"
- p≥α → 不能拒绝零假设，差异"不显著"
- 不存在"边缘显著""轻微显著""趋势显著" → 这些是p值误用
- 如果p接近阈值 → 报告"不显著，但建议增加样本量复核"，而非"边缘显著"
- 阈值必须在分析前设定，事后不可更改

### Q4: 这个场景该用什么检验？

**Hints:**
- 思考"同一测试集的不同折"意味着什么
- 独立样本t检验的假设：两组样本独立
- 配对t检验的假设：配对差异独立

**Answer:**
**必须用配对t检验。**

10折交叉验证中，每一折的测试数据有重叠（同一测试集的不同划分），旧模型和新模型在每一折上的AUC评估基于**相同的测试样本**。这意味着两组AUC不是独立的——如果某一折碰巧包含"难样本"，两个模型的AUC都会低；如果包含"易样本"，两个AUC都会高。

独立样本t检验假设两组样本独立，违反这一假设会导致：
1. 方差被高估（因为忽略了配对正相关）
2. t值被低估，p值被高估
3. 可能把"真实差异"判断为"不显著"

配对t检验计算每一折的"新模型AUC - 旧模型AUC"差异，检验差异均值是否为零。它利用了"同一折内正相关"的信息，方差更小，检验效能更高。

**决策规则：**
- 两组数据来自**不同样本** → 独立样本t检验
- 两组数据来自**同一样本的两次测量**（前后测、配对实验、交叉验证的不同模型） → 配对t检验
- 不确定是否配对 → 问"去掉一个样本，另一组是否也少一个样本？"如果是，则配对
- 用错检验 → 独立t检验用于配对数据会损失检验效能；配对t检验用于独立数据会低估方差

### Q5: p<0.001但效应量几乎为零，该全量吗？

**Hints:**
- 思考p值和效应量分别回答什么问题
- Cohen's d=0.02在业务上意味着什么？
- "统计显著"和"业务显著"有什么区别？

**Answer:**
**不一定全量。** p值回答"差异是否真实"，效应量回答"差异是否重要"。p<0.001说明"差异极不可能由随机波动造成"，但Cohen's d=0.02说明"差异极小，几乎无实际意义"。

美团案例：点击率提升0.3pp，日均多1000次点击，但后续转化无变化。这意味着：
1. 效应真实存在（非随机），但极小
2. 开发和QA成本 > 收益
3. 全量后用户体验可能无感知，但系统复杂度增加

**统计显著 vs 业务显著：**
- 统计显著：差异"真实"（非随机），由大样本驱动
- 业务显著：差异"重要"（有价值），由效应量驱动
- 大样本下，极小的效应也能p<0.001——这是"统计显著"的陷阱

**决策规则：**
- 实验报告必须同时报告：p值 + 效应量（Cohen's d或原始效应）+ 置信区间
- Cohen's d<0.2（小效应） → 即使p<0.001，也需评估ROI
- 决策公式：预期收益 = 效应量 × 用户规模 × 单位价值 - 实施成本
- 如果预期收益 < 实施成本 → 不采纳，即使"统计显著"
- 向业务方解释："差异真实但极小，全量的收益覆盖不了成本"
