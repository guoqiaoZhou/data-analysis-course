# 固定效应模型

## 基本原理

固定效应模型（Fixed Effects Model）用于面板数据（同一单位多个时间点的观测），通过控制不随时间变化的个体特征来消除遗漏变量偏误。

**核心思想：**
- 每个个体有一个"固定"的特质（如用户偏好、商家能力），不随时间变化
- 通过个体虚拟变量或去均值化（Demeaning），消除这些特质的影响
- 只利用个体**内部**的时间变化来估计效应

**模型形式：**
$$Y_{it} = \alpha_i + \beta X_{it} + \epsilon_{it}$$

其中 $\alpha_i$ 是个体固定效应，可以是虚拟变量或去均值化后的截距。

## 使用场景

| 场景 | 说明 | 示例 |
|---|---|---|
| 面板数据分析 | 同一用户/商家多期数据 | 评估会员制度对用户消费的影响 |
| 消除不随时间变化的混杂 | 控制个体固有特征 | 控制用户"天生偏好"后评估推荐效果 |
| 政策评估（准实验） | 处理在不同时间对不同个体实施 | 某城市先实施新规，评估对GMV的影响 |
| 前后测比较 | 同一组处理前后的变化 | 功能改版前后的用户留存变化 |

## 局限性

1. **只能控制不随时间变化的混杂**：随时间变化的混杂仍需其他方法
2. **无法估计不随时间变化的变量**：如性别、城市固定特征
3. **要求面板数据**：需要同一单位多期观测，流失用户无法使用
4. **序列相关**：误差项可能存在自相关，标准误需聚类调整
5. **时间趋势**：需控制共同时间趋势，否则混淆处理效应

## 同类算法对比

| 方法 | 适用条件 | 优势 | 劣势 |
|---|---|---|---|
| **固定效应（FE）** | 面板数据，不随时间变化的混杂 | 消除个体固定特质 | 无法估计时不变变量 |
| **随机效应（RE）** | 面板数据，个体效应与自变量不相关 | 更 efficient，可估计时不变变量 | 假设强，通常不成立 |
| **混合OLS（Pooled OLS）** | 面板数据，无个体效应 | 简单 | 遗漏变量偏误 |
| **DID** | 处理在不同时间对不同组 | 利用对照组控制共同趋势 | 需要平行趋势假设 |
| **一阶差分（FD）** | 面板数据，T=2 | 简单直观 | 损失信息，T>2时不如FE |

## 真实案例

### 案例1：会员制度效果评估（C端运营）

**背景：** 评估"付费会员"对用户月均消费的影响，用户在不同时间开通会员。

**数据：** 10000用户 × 12个月的面板数据

**模型：**
```
月均消费_{it} = α_i + β×会员_{it} + γ×月份虚拟变量 + ε_{it}
```

**分析：**
- 固定效应 α_i 控制用户固有消费水平
- β = +85（p<0.001），会员用户月均消费高85元
- 但注意：可能是高消费用户更可能开会员（自选择），需进一步用DID或工具变量

### 案例2：商家运营策略效果（B端运营）

**背景：** 评估"运营经理对接"对商家GMV增长的影响，不同商家在不同时间获得对接。

**数据：** 5000商家 × 24个月

**模型：**
```
GMV_{it} = α_i + β×对接_{it} + δ×季度趋势 + ε_{it}
```

**分析：**
- 固定效应控制商家固有规模、行业、地域
- β = +12%（p=0.003），对接后GMV增长12%
- 局限：无法区分是运营经理的作用，还是商家发展到一定阶段的自然增长

### 案例3：功能改版评估（离线分析）

**背景：** 评估搜索功能改版对用户搜索频次的影响。

**数据：** 同一批用户改版前后各30天的搜索频次

**模型（一阶差分）：**
```
Δ搜索频次_i = β×改版_i + Δε_i
```

**分析：**
- 用每个用户改版前后的差分，消除用户固有搜索习惯
- β = +0.5次/天（p<0.05），改版后日均搜索增加0.5次
- 但样本仅保留改版前后都活跃的用户，可能幸存者偏误

## 关键词

- **面板数据**：同一单位（用户/商家）多个时间点的观测
- **固定效应**：不随时间变化的个体特质，α_i
- **去均值化（Demeaning）**：$Y_{it} - \bar{Y}_i$，消除个体均值
- **组内变异**：个体随时间的变化，FE只利用这部分信息
- **组间变异**：个体之间的差异，FE不利用
- **Hausman检验**：检验FE vs RE的选择，若拒绝则选FE
- **聚类标准误**：同一单位多期观测误差相关，需按个体聚类

## 行业使用经验

### 经验1：骑手补贴政策评估——固定效应剥离个体固有效率

美团外卖2021年Q3评估"雨天加价补贴"对骑手接单率的影响。初始OLS显示补贴使接单率提升20%（p<0.001），但业务方质疑：高能力骑手更可能在雨天出工，也更容易获得补贴。引入骑手固定效应后，政策效应从+20%降至+5%（p=0.02）。固定效应控制了骑手固有效率（如熟悉路况、体力水平等不随时间变化的因素），剩余+5%才是补贴的真实因果效应。这个案例展示了FE在消除不随时间变化的自选择偏误上的威力。

### 经验2：FE无法消除时变自选择——临时收入冲击的案例

某网约车平台2022年用FE评估"高峰奖励"对司机出车时长影响，FE估计显示奖励使日工作时长+1.2小时。但进一步分析发现：司机在获得临时大额订单（非奖励因素）后，未来三天出车时长系统性增加。这是时变自选择——临时收入冲击同时影响"是否获得奖励"和"未来出车行为"，FE无法消除。后引入"前一日收入"作为控制变量，效应降至+0.4小时。FE不是万能工具，时变混杂仍需额外处理。

### 经验3：Hausman检验功效低，实际应结合理论判断

某内容平台2023年比较FE和RE模型，Hausman检验p=0.08（不拒绝RE）。团队最初选择RE，因为更efficient且可估计时不变变量。但业务逻辑上：用户固定效应（如天生内容偏好）几乎必然与处理变量（如推荐策略暴露）相关。后改用FE，核心系数方向一致但置信区间更宽。 Hausman检验在面板数据T较小时功效不足，容易错误接受RE。我们的经验：当理论上个体效应与自变量相关时，直接选FE，Hausman检验仅作参考。

### 经验4：我们曾犯的错误——把FE当万能工具，忽略了时间趋势控制

2020年我们评估某功能改版对用户留存的影响，用FE模型控制了用户固定效应，结果显示改版使留存+8%（p<0.01）。但改版恰好发生在春节后——用户整体留存季节性回升。未控制时间趋势的FE模型将季节效应误归于改版。加入月份虚拟变量后，效应降至+2%且不显著。FE消除了个体间差异，但个体共同经历的时间趋势仍需显式控制。此后我们规定：任何FE模型必须同时包含时间固定效应（Two-way FE），除非有明确理由省略。
