# 好问练习解答 - 固定效应模型

> 对应知识点：[[stage-01-foundation/02-statistical-basics/06-fixed-effects/固定效应模型|固定效应模型]]

---

### Q1: 为什么同样的数据，加了固定效应后政策效应从+20%降到了+5%？

**Hints:**
- 思考OLS估计中"获得补贴的骑手"和"未获得补贴的骑手"之间，除了补贴还有什么系统性差异
- 固定效应模型利用的是"组内变异"还是"组间变异"？
- 骑手固定效应 α_i 控制了哪些不可观测的因素？

**Answer:**
OLS的+20%是"获得补贴的骑手 vs 未获得补贴的骑手"的比较，混杂了骑手固有特质（如勤快程度、熟悉路况、体力水平）。高能力骑手更可能在雨天出工，也更容易获得补贴——这是不随时间变化的自选择偏误。

固定效应模型通过控制骑手固定效应 α_i，将比较基准变为"同一个骑手获得补贴时 vs 未获得补贴时"，消除了骑手间固有差异。剩余的+5%才是补贴的真实因果效应。

**决策规则：**
- 如果数据是面板数据（同一单位多期观测），且存在不随时间变化的遗漏变量 → 用固定效应
- 如果OLS估计的效应远大于FE估计 → 提示存在严重的个体自选择偏误
- 如果FE估计不显著而OLS显著 → 说明"组间差异"是OLS显著的主要来源，处理效应本身可能很弱

### Q2: 固定效应模型和随机效应模型，到底该用哪个？

**Hints:**
- Hausman检验的原假设是什么？它的功效受什么因素影响？
- 从业务逻辑上，商家固有特质与"是否获得运营对接"是否可能独立？
- RE模型的核心假设是什么？这个假设在真实业务中通常成立吗？

**Answer:**
**选固定效应（FE）。**

RE模型的核心假设是个体效应 α_i 与自变量 X_it 不相关。在业务场景中，这个假设几乎必然被违反：
- 高GMV商家更可能获得运营经理对接（商家规模与处理相关）
- 高消费用户更可能开通会员（用户偏好与处理相关）
- 勤快骑手更可能获得雨天补贴（骑手特质与处理相关）

Hausman检验在面板数据T较小时功效不足，容易错误接受RE。理论上个体效应与自变量相关时，直接选FE，Hausman检验仅作参考。

**决策规则：**
- 如果理论上个体效应与处理变量相关 → 直接选FE，不看Hausman检验
- 如果个体效应与处理变量确实独立（罕见）→ 可用RE，获得更窄的置信区间
- 如果需要估计不随时间变化的变量（如性别、城市）→ 考虑FE+HT（Hausman-Taylor）或 Mundlak 方法

### Q3: 为什么我的固定效应模型把春节后的季节性回升误当成了改版效应？

**Hints:**
- 固定效应消除了"个体间差异"，但消除了"个体共同经历的时间变化"吗？
- 如果所有用户同时在春节后留存回升，固定效应能否区分这是时间趋势还是改版效应？
- Two-way FE（个体固定效应+时间固定效应）能解决什么问题？

**Answer:**
固定效应模型（仅个体FE）消除了不随时间变化的个体特质，但没有控制所有个体共同经历的时间趋势。当改版时间与季节性趋势（如春节后自然回升）重合时，模型将时间趋势误归于改版效应。

**正确做法：** 使用Two-way Fixed Effects（双向固定效应），同时控制个体固定效应和时间固定效应（月份/季度虚拟变量）：

$$Y_{it} = \alpha_i + \lambda_t + \beta X_{it} + \epsilon_{it}$$

其中 λ_t 控制所有个体在第t期的共同冲击。

**决策规则：**
- 任何FE模型必须同时包含时间固定效应，除非有明确理由省略
- 如果处理时机与已知季节性/事件重合 → 必须控制时间FE
- 加入时间FE后效应大幅下降 → 说明原估计中混杂了时间趋势

### Q4: 这个场景该用固定效应还是DID？

**Hints:**
- 固定效应的核心比较是什么？DID的核心比较是什么？
- 场景A中所有司机在同一时间经历政策变化，存在"时间趋势"和"政策效应"的完全共线性问题吗？
- 场景B中不同城市在不同时间实施政策，这提供了什么额外信息？

**Answer:**

**场景A（所有司机同一时间经历政策变化）：固定效应无法使用，或只能做一阶差分。**

如果所有个体在同一时间接受处理，处理变量 X_it 与时间趋势完全共线性，个体FE+时间FE会吃掉处理效应。此时只能：
- 用一阶差分（前后测比较），但无法区分政策效应与同期其他冲击
- 或承认设计缺陷，改用其他方法

**场景B（城市分批实施政策）：用DID或双向FE。**

 staggered DID 利用"已实施城市 vs 未实施城市"的差异，控制共同时间趋势。不同实施时间提供了处理组和对照组随时间变化的对比，解决了共线性问题。

**决策规则：**
- 如果所有个体同一时间接受处理 → FE无法分离时间趋势，考虑DID（需有对照组）或承认设计局限
- 如果个体分批接受处理 → 用 staggered DID 或 event study
- 如果只有处理组无对照组 → FE一阶差分，但结论弱

### Q5: 骑手固定效应控制了"固有勤快程度"，但为什么临时收入冲击仍然是个问题？

**Hints:**
- 固定效应能控制什么类型的混杂？不能控制什么类型的混杂？
- "临时收入冲击"是随时间变化的还是不随时间变化的？
- 引入"前一日收入"作为控制变量，可能产生什么新的问题？

**Answer:**

**固定效应只能消除不随时间变化的混杂，无法消除时变混杂。** "临时收入冲击"是随时间变化的——骑手某天接到大单，未来三天行为改变。这种时变自选择FE无法处理。

**三个选项的评估：**

| 选项 | 优劣 | 建议 |
|---|---|---|
| 引入前一日收入作为控制 | 可能过度控制（收入是行为结果，控制它会堵住部分因果路径） | 不推荐作为核心方案 |
| 工具变量法 | 若找到有效IV（如"平台随机派单优先级"），可解决内生性 | 首选，但IV需严格论证外生性 |
| 接受+1.2小时为上界 | 诚实声明估计偏误方向（上界），不假装精确 | 若前两种不可行，这是负责任的方案 |

**决策规则：**
- 如果存在合理的时变混杂且能找到外生IV → 用IV/2SLS
- 如果无法找到IV → 在报告中明确声明"FE估计在时变混杂存在时为效应上界"
- 永远不要假装FE已经解决了所有内生性
