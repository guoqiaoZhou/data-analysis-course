# 好问练习 - 检验力分析与样本量计算

> 对应知识点：[[stage-01-foundation/02-statistical-basics/07-power-analysis/检验力分析与样本量计算|检验力分析与样本量计算]]

---

## Q1: 产品经理说"最好能检测到0.1%的点击率变化"，你算了算说需要200万用户/组，他立刻改口说"那0.5%吧"——这个对话背后的统计逻辑是什么？ [L1 概念层]

你是某电商平台的数据分析师。产品团队要做新推荐策略的AB实验，PM问你样本量需求。你反问MDE，他说越小越好。你展示了样本量公式 n = 16σ²/δ² 的计算过程：检测0.1%变化需200万/组，检测0.5%需8万/组。PM从"越小越好"瞬间变成"0.5%也行"。你需要理解并向团队解释：为什么样本量对MDE如此敏感？以及"能检测"和"值得检测"之间的区别。

## Q2: 样本量公式 n = 16σ²/δ² 和精确公式 n = 2[(Z₁₋α/₂ + Z₁₋β)σ/δ]²，到底该用哪个？ [L2 混淆层]

你在规划一个AB实验，需要向团队汇报样本量。你记得"16法则"可以快速估算，但同事坚持用精确公式计算。两个公式算出来的结果差异有多大？什么情况下"16法则"会严重误导？更根本的问题是：这两个公式背后的假设（正态分布、独立样本、等方差）在真实业务数据中成立吗？

## Q3: 实验跑完了不显著，我说"可能是样本量不够"，老板反问"那你当初为什么没算对？"——哪里出了问题？ [L3 错误层]

你规划了一个新功能实验，按历史数据方差计算样本量，每组5万用户。实验跑了2周，效应不显著。复盘发现：实验期恰逢春节大促，转化率方差比历史"平稳期"数据高了80%。你当初用"最佳月份"的方差计算样本量，结果实验期赶上高方差时段，灵敏度不足。老板质疑你的专业能力。这个错误为什么如此常见？为什么"历史方差"不等于"实验期方差"？

## Q4: 同时监控5个指标，样本量该怎么调整？ [L4 应用层]

你设计了一个AB实验，核心指标是转化率，同时监控4个护栏指标（点击率、停留时长、退款率、复购率）。按单指标计算每组需2万用户。但同事提醒：同时检验5个指标，整体假阳性风险从5%飙升到23%（1-0.95⁵）。你现在面临选择：
- 选项A：Bonferroni校正，α=0.01/指标，样本量增加
- 选项B：不校正，接受更高假阳性风险
- 选项C：只关注核心指标，护栏指标仅做探索性分析

你会怎么选？判断标准是什么？

## Q5: 业务方说"变化<1pp不值得动"，统计方说"按1pp算需要跑6周"——MDE到底谁说了算？ [L5 综合层]

某内容平台实验，业务方期望检测"人均停留时长+5秒"（认为有意义）。历史σ=120秒，计算得n=460,000/组，需跑6周。统计方认为+5秒/120秒=4.2%的相对效应，在业务上"可检测"但"不值得等6周"，建议MDE=+15秒（n=51,000/组，1周完成）。业务方坚持"5秒有意义"，统计方坚持"6周不现实"。你作为分析师，如何促成双方对齐？MDE的本质是统计问题还是业务问题？
