# 好问练习解答 - 卡方检验

> 对应知识点：[[stage-01-foundation/02-statistical-basics/02-chi-square-test/卡方检验|卡方检验]]

---

### Q1: 运营说"男用户爱买数码，女用户爱买美妆"，我该怎么证明这不是巧合？

**Hints:**
- 思考"如果性别和品类真的没关系"，理论上的频数应该怎么计算？
- 观测值和理论值的差距，大到什么程度才不算"抽样波动"？

**Answer:**
用卡方独立性检验。核心逻辑：假设性别和品类独立，计算每个单元格的期望频数 $E_{ij} = \frac{(行合计_i)(列合计_j)}{总样本量}$，然后计算 $\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$。如果观测值与期望值的偏差大到卡方分布的右尾（p<0.05），则拒绝"独立"假设，认为存在真实关联。

**决策规则：**
- 如果所有期望频数≥5且总样本量>40，用Pearson卡方检验
- 如果有20%以上单元格期望频数<5，用Fisher精确检验或合并类别
- 卡方检验显著后，必须报告Cramér's V效应量，不能只报p值

### Q2: 卡方检验和t检验看起来都是"检验差异是否显著"，面对"两组转化率不同"该用哪个？

**Hints:**
- t检验的"均值"在0-1数据上是什么意思？
- 当样本量很大时，t检验对比例数据的近似是否可靠？

**Answer:**
两组比例比较时，t检验和卡方检验在数学上是等价的（卡方统计量等于Z统计量的平方），大样本下结论一致。但选择方法应基于数据类型而非便利性：

**决策规则：**
- 如果因变量是0/1编码，自变量是二分类 → 卡方检验更自然，因为卡方就是为分类变量设计的
- 如果自变量是多分类（三组及以上转化率比较） → 只能用卡方检验，t检验无法直接处理
- 如果需要报告效应量或置信区间 → 用比例差异的Z检验或Logistic回归，卡方检验本身不提供置信区间
- 如果数据是连续变量的均值比较 → 用t检验，卡方检验不适用

**本质区别：** t检验处理连续变量的位置参数；卡方检验处理分类变量的联合分布结构。两者在2×2表上数值等价，但解释框架不同。

### Q3: 卡方检验p<0.001，我兴奋地宣布"版本A最优"并全量上线，三个月后留存回到原点——我错在哪？

**Hints:**
- 卡方检验的备择假设是什么？是"A最优"还是"不全相同"？
- 从三组中事后挑出最好的一组，和事先指定一组做检验，错误率有什么区别？

**Answer:**
你犯了三个错误：

1. **把"不全相同"当成"A最优"**：卡方检验的备择假设是"至少有一组不同"，不是"A组最好"。显著只说明三组不全相等，不说明哪组最优。

2. **事后多重比较未校正**：从三组中挑表现最好的做决策，相当于做了多次检验。Bonferroni校正要求将α除以比较次数（3组两两比较需α/3=0.017），未校正时第一类错误膨胀到约14%。

3. **忽略样本量不均**：版本A样本量大20%，使其"看起来"更好可能只是抽样波动被放大。

**决策规则：**
- 卡方检验显著后，如需判断"哪组最优"，必须进行事后多重比较（Bonferroni或FDR控制）
- 如果目的是排序，直接用比例差异的置信区间，不用卡方检验
- 样本量不均时，报告加权效应量，不直接比较原始比例

### Q4: 样本量10万时卡方检验p<0.001，但效应量只有0.02——我该信哪个？

**Hints:**
- p值和样本量是什么关系？固定效应量下，样本量翻倍p值会怎样？
- Cramér's V=0.02在实际业务中意味着什么级别的差异？

**Answer:**
信效应量。p值同时受效应大小和样本量影响：$p \approx f(\text{效应量}, \sqrt{n})$。样本量10万时，即使微不足道的关系也会被"显著"。Cramér's V=0.02属于可忽略效应（<0.1），意味着性别对品类偏好的解释力几乎为零。

**决策规则：**
- 如果p<0.05但Cramér's V<0.1 → 报告"统计显著但业务上不相关，不建议投入"
- 如果p>0.05但Cramér's V>0.3 → 检查样本量是否不足，可能需要扩大样本重测
- 业务决策必须以效应量为准，p值只用于判断"不是随机波动"
- 向老板解释："这个差异确实存在，但小到用户几乎感知不到，投入产出比不划算"

### Q5: 交叉表分析时，卡方检验显著、Fisher精确检验不显著、G检验边缘——三个方法打架，我听谁的？

**Hints:**
- 卡方检验的"大样本近似"在什么条件下成立？
- Fisher精确检验的"精确"指的是什么？它的保守性从何而来？

**Answer:**
听Fisher精确检验。方法选择应基于数据特征而非结果偏好：

**决策规则：**
- 样本量<1000或期望频数<5的单元格>20% → 优先用Fisher精确检验，因为它是精确计算而非近似
- 大样本、无稀疏单元格 → 卡方检验和G检验均可，结果应基本一致
- 如果三种方法结论矛盾 → 说明数据处于"边界状态"，结论本身就不稳定，应报告"边缘显著，建议扩大样本"

**关键原则：**
- 方法选择必须在看结果之前决定，事后换方法等于p值操控
- 建立预分析计划：n<1000用Fisher，n≥1000用卡方，期望频数检查是必做步骤
- 如果结论对方法选择敏感，结论本身不可信，需要更多数据或更精细的分析
