# 好问练习解答 - OLS回归

> 对应知识点：[[stage-01-foundation/02-statistical-basics/04-ols-regression/OLS回归|OLS回归]]

---

### Q1: 运营问"配送费每涨1块，订单量降多少"，我凭什么画这条线而不是另一条？

**Hints:**
- "最好"的直线需要什么数学性质？垂直距离的平方和最小，为什么不用绝对值？
- 如果数据点不在一条直线上，任何直线都有误差，怎么量化"总误差"？

**Answer:**
用OLS（普通最小二乘法）。核心标准：找到使所有数据点到直线的垂直距离（残差）的**平方和最小**的直线。

**为什么用平方和而非绝对值：**
- 平方对大的偏差惩罚更重，避免极端值被多个小偏差抵消
- 平方函数可导，有解析解；绝对值在零点不可导，优化更复杂
- 在高斯-马尔可夫假设下，OLS估计量是BLUE（最佳线性无偏估计）

**解析解：**
$$\hat{\beta} = (X'X)^{-1}X'Y$$

**决策规则：**
- 如果关系近似线性、误差近似正态、无强异常值 → OLS是首选
- 如果有强异常值 → 考虑稳健回归（Huber、Tukey）或删除异常值后重跑
- 如果目的是预测而非解释 → 可考虑正则化方法（Ridge、Lasso）

### Q2: OLS回归和Ridge回归都能拟合线性关系，面对"配送费和距离高度相关"该用哪个？

**Hints:**
- VIF=18意味着什么？系数的标准误会被放大多少倍？
- Ridge回归的"有偏"和OLS的"不稳定"，哪个对业务决策伤害更大？

**Answer:**
根据分析目的选择方法：

| 目的 | 推荐方法 | 理由 |
|------|----------|------|
| **解释因果关系** | 处理多重共线性后OLS | 系数解释必须无偏，VIF>10的变量需删除或合并 |
| **预测为主** | Ridge回归 | 有偏但稳定，预测MSE通常更低 |
| **两者兼顾** | 主成分回归或偏最小二乘 | 保留解释力的同时降低共线性 |

**决策规则：**
- VIF>10 → 必须处理：删除冗余变量、做PCA、或用正则化
- 如果"距离"和"配送费"高度相关 → 保留业务上更直接的那个（通常是配送费），删除距离
- 删除变量后重新检查VIF矩阵，直到所有VIF<5
- 如果两个变量都重要且不能删 → 报告它们的"综合效应"（如构造交互项或加权指数），不单独解释各自系数

**关键原则：** 多重共线性下OLS系数"可以估计但无法解释"——标准误膨胀导致t检验失效，系数符号可能反转。此时OLS不是"有点问题"，而是"危险"。

### Q3: R²=0.42，团队庆祝"模型很好"，但业务方问"那剩下的58%是什么"——我错在哪？

**Hints:**
- R²高是否意味着模型"好"？残差图有模式是否意味着模型"坏"？
- 漏斗形残差（异方差）对系数的估计和标准误分别有什么影响？

**Answer:**
你犯了三个错误：

1. **把R²当质量指标**：R²只是"模型解释了Y变异的多少比例"，不是"模型是否正确"。一个错误的模型（如遗漏关键变量）也可以有很高的R²。

2. **忽略残差诊断**：残差图呈漏斗形说明异方差——误差方差随预测值变化，违反同方差假设。此时OLS系数估计仍然无偏，但标准误无效，t检验和置信区间不可靠。

3. **未报告残差图**：任何回归分析必须附残差图（残差vs拟合值、Q-Q图）。若残差有模式，R²再高也不可信。

**决策规则：**
- 报告R²时必须同时展示残差图
- 残差呈漏斗形 → 使用White稳健标准误（HC3），或做方差稳定变换（如对数变换）
- 残差有曲线模式 → 模型设定错误，需加入非线性项（多项式、样条）或遗漏变量
- 残差有异常值 → 检查数据质量，考虑稳健回归

### Q4: 观察性数据中OLS系数显著为正，产品团队据此全量推广——上线后效果平平，我该信系数还是信常识？

**Hints:**
- OLS的外生性假设 $E[\epsilon|X]=0$ 在"用户自主选择是否使用搜索"时是否成立？
- 高活跃用户更可能搜索，这个"选择性"进入了误差项还是自变量？

**Answer:**
信常识。观察性数据中的OLS系数**永远只是条件相关，不是因果效应**。外生性假设被违反时，系数是有偏的——"搜索"的系数吸收了"用户活跃度"的效应，因为活跃度在误差项中且与搜索相关。

**决策规则：**
- 观察性数据 + 自变量非随机分配 → OLS系数不可解释为因果，只能描述"使用过搜索的用户留存更高"
- 需要因果结论 → 必须做随机实验、工具变量、或DAG支持的因果识别策略
- 如果无法实验 → 用倾向得分匹配（PSM）或双重差分（DID）降低选择偏误，但仍不如随机实验可靠
- 向业务方解释："数据显示搜索用户留存高15%，但这可能是因为活跃用户更爱搜索。要确认搜索本身的效果，需要AB实验。"

### Q5: 残差图显示异方差，普通标准误p=0.04（"显著"），White稳健标准误p=0.11（"不显著"）——我该按哪个上报？

**Hints:**
- 异方差下普通标准误的性质是什么？它是"有点不准"还是"系统性地错"？
- White标准误牺牲了效率换取稳健性，这个 trade-off 在什么情况下值得？

**Answer:**
按White稳健标准误上报。异方差下普通标准误**系统性地偏误**（可能高估或低估，取决于异方差模式），不是"有点不准"。White（1980）证明：异方差一致性标准误（Heteroskedasticity-Consistent Standard Errors）在大样本下仍然有效，虽然效率略低。

**决策规则：**
- 任何OLS回归，**默认使用稳健标准误**（HC3），这是现代计量经济学的标准做法
- 在Python的statsmodels中：`sm.OLS(y, X).fit(cov_type='HC3')`
- 在R中：`coeftest(model, vcov = vcovHC(model, type = "HC3"))`
- 如果普通标准误和稳健标准误结论一致 → 报告稳健版本，增加可信度
- 如果两者矛盾 → 以稳健版本为准，并解释异方差的存在

**关键原则：** "默认稳健"应该是分析流程的硬性规定，不是"发现异方差后再补救"。事后补救等于在p值边缘反复试探，是p值 hack 的一种形式。
