# 好问练习解答 - 观察性数据的挑战

> 对应知识点：[[stage-01-foundation/01-causal-inference-basics/03-observational-challenges/观察性数据的挑战|观察性数据的挑战]]

---

### Q1: 为什么"使用高级报表的客户续约率高30%"不能证明报表功能有价值？

**Hints:**
- 思考"客户投入度"如何同时影响"使用报表"和"高续约率"
- 回忆混杂变量的定义：同时影响处理和结果的变量
- 如果强制让低投入客户使用报表，他们的续约率会提高吗？

**Answer:**
"使用报表"和"高续约率"的相关性由混杂变量"客户投入度"驱动：高投入客户更可能使用报表（处理），也更可能续约（结果）。这不是"报表→续约"的因果路径，而是"投入度→报表"和"投入度→续约"的共同结果。

观察性数据的核心困境是：处理分配不是随机的，而是由个体特征、环境等因素决定。$D \not\perp (Y(1), Y(0))$，简单比较处理组和对照组得到的是"选择性偏误 + 因果效应"的混合，无法分离。

**决策规则：**
- 发现"A用户做B事效果更好" → 先问"是B导致好效果，还是好用户才做B？"
- 如果存在未控制混杂（如用户投入度、活跃度、消费能力） → 相关性≠因果
- 观察性研究的因果结论必须附带：方法假设、敏感性分析、未观测混杂评估
- 任何基于观察性数据的策略建议 → 优先用实验验证，或明确标注"待验证假设"

### Q2: 回归调整、匹配、倾向得分——三种控制混杂的方法有什么区别？

**Hints:**
- 思考回归系数"控制X后D的效应"在数学上做了什么
- 匹配是在"选择样本"还是"调整估计方法"？
- IPW的权重是如何"模拟随机化"的？

**Answer:**
**回归调整**：在模型中加入协变量X，用线性（或广义线性）关系"剥离"X对Y的影响，估计D的净效应。数学上：$Y = \alpha + \tau D + \beta X + \epsilon$，$\tau$ 是"控制X后D的效应"。失效条件：模型误设（真实关系非线性）、X与D高度共线、遗漏混杂。

**匹配（PSM）**：为每个处理组单位找到"相似"的对照组单位（基于协变量或倾向得分），构建"伪随机化"样本。数学上：用 $P(D=1|X)$ 综合所有协变量，在得分相近的单位间比较。失效条件：共同支撑假设不满足（处理组和对照组X分布无重叠）、匹配后样本损失大、只处理观测混杂。

**IPW（逆概率加权）**：给每个样本赋权 $w = 1/P(D=1|X)$（处理组）或 $1/P(D=0|X)$（对照组），使加权后的样本在X分布上"看起来像随机化"。失效条件：倾向得分极端（接近0或1，导致权重爆炸）、倾向得分模型误设。

**决策规则：**
- 协变量少（<10个）、关系近似线性 → 回归调整简单有效
- 协变量多、需要可视化平衡 → PSM更直观，可检查共同支撑
- 大样本、倾向得分可准确估计 → IPW或双重稳健（DR）更优
- 任何方法都只处理**观测混杂** → 未观测混杂仍需敏感性分析或工具变量

### Q3: 为什么"控制所有变量"是观察性研究最大的陷阱？

**Hints:**
- 思考"配送经验"在"培训→准时率"路径中的位置
- 回忆中介变量的定义：位于处理与结果之间的变量
- 控制中介变量后，回归系数估计的是什么？

**Answer:**
"配送经验"是培训→准时率的中介变量：培训提高了骑手的配送经验，而经验提高又提升了准时率。控制中介变量等于切断了真实的因果路径，回归系数只估计"培训不通过经验的直接效应"——这不是总效应，而是被低估的偏效应。

初学者犯"控制所有变量"错误的原因：
1. 直觉上"控制越细越准"，但不知道变量在因果图中的位置决定其是否该控制
2. 统计软件不区分混杂和中介，只要放进模型就"控制"
3. 业务方要求"多控制一些"，分析师不加判断照做

**决策规则：**
- 任何观察性分析 → **先画DAG（有向无环图）**，再决定控制什么
- 混杂变量（同时影响处理和结果）→ **必须控制**
- 中介变量（处理→中介→结果）→ **绝不能控制**
- 碰撞变量（被处理和结果共同影响）→ **绝不能控制**（控制会打开虚假路径）
- 不确定变量位置 → 用DAG工具（如dagitty.net）辅助判断

### Q4: 这个场景该用什么方法处理混杂？

**Hints:**
- 思考DID的"平行趋势"假设在这个场景中为什么难满足
- 工具变量的"相关性"和"排他性"约束，弹窗推送满足吗？
- PSM的局限是什么？它能解决"消费意愿"这种未观测混杂吗？

**Answer:**
**优先方法A（PSM），但必须附带敏感性分析。**

方法C（DID）不适用：会员开通时间分散，开通前趋势各不相同，"平行趋势"假设几乎不可能满足。DID要求处理组和对照组在没有处理的情况下趋势平行，但会员和非会员的消费趋势天然不同。

方法B（工具变量）风险高：弹窗推送由算法驱动，算法可能基于用户特征（如活跃度、消费能力）推送——这意味着工具变量与结果相关（违反排他性约束）。"被推送弹窗"不纯粹是随机冲击，混杂了用户特征。

方法A（PSM）的局限：只能处理观测混杂，"消费意愿"等未观测因素仍可能偏倚结果。但PSM可处理高维观测混杂，且可检查共同支撑和平衡性。

**决策规则：**
- 有面板数据+平行趋势可信 → DID优先
- 有真正随机或准随机的工具变量 → 工具变量法优先
- 只有截面数据+高维协变量 → PSM或IPW，但必须做敏感性分析
- 方法选择取决于**数据结构和假设可信度**，而非"哪个方法更高级"
- 任何观察性结论 → 必须回答"未观测混杂可能多大？"才能被业务采纳

### Q5: 未观测混杂存在时，观察性研究的结论还能用吗？

**Hints:**
- 思考Rosenbaum bounds中"Γ=2.5"的业务含义
- 什么程度的未观测混杂会"推翻"当前结论？
- 业务决策中"确定性"和"行动紧迫性"如何权衡？

**Answer:**
**可以谨慎使用，但必须明确标注置信度。**

Rosenbaum bounds中Γ=2.5意味着：需要一个使开会员odds ratio > 2.5的未观测变量才能解释掉300元/年的效应。换句话说，这个未观测混杂必须非常强——比大多数已知混杂（如年龄、收入、历史消费）的影响还要大——才能推翻结论。

业务采纳标准：
1. **敏感性分析门槛**：Γ > 2通常被认为"稳健"（需未观测混杂非常强才能推翻）
2. **领域知识**：该领域是否存在已知强混杂未被纳入？如存在，门槛需提高
3. **决策成本**：如果决策成本低（如UI调整），可接受较低置信度；如果决策成本高（如定价策略），需更高置信度
4. **补充验证**：能否设计小规模实验验证？能否用负对照检验？

**决策规则：**
- Γ < 1.5 → 结论不稳健，不建议采纳
- Γ = 1.5-2.0 → 中等稳健，可采纳但需标注"观察性研究，存在未观测混杂风险"
- Γ > 2.0 → 相对稳健，可采纳但建议补充实验验证
- 任何观察性因果结论 → 必须附带：方法、假设、敏感性分析结果、置信度声明
- 高成本决策 → 即使Γ>2.0，也应优先用实验验证
